8.2 First-Order Methods

First-order methods compute first-order derivatives of the objective function but they ignore the curvature information.

8.2.2 Step size

The sequence of steps {ηt}\{\eta_t\}{ηt​}﻿ is the learning rate schedule. We describe some methods to pick this.
8.2.2.1 Constant step size
The simplest method is to consider ηt=η\eta_t=\etaηt​=η﻿. However, if set too large the method will fail to converge, and if set too small, the method will converge very slowly.
We can derive a theoretical upper bound for the step size. For a quadratic objective function:
L(θ)=12θ⊤Aθ+b⊤θ+c\mathcal{L}(\theta)=\frac{1}{2}\theta^\top A \theta+\bold{b}^\top\theta+cL(θ)=21​θ⊤Aθ+b⊤θ+c
with AA A﻿ positive semi-definite, the steepest descent will converge iff the step size satisfies:
η<2λmax(A)\eta <\frac{2}{\lambda_{max}(A)}η<λmax​(A)2​
This ensures that the step is lower than the slope of the steepest direction, which the eigenvalue measures.
More generally:
η<2L\eta <\frac{2}{L}η<L2​
where LLL﻿ is the Lipschitz constant of the gradient —but this constant is usually unknown.

8.2.3 Convergence rates

In some convex problems where the gradient is bounded by the Lipschitz constant, gradient descent converges at a linear rate:
∣∣L(θt+1)−L(θ∗)∣∣≤μ∣∣L(θt)−L(θ∗)∣∣||\mathcal{L}(\theta_{t+1})-\mathcal{L}(\theta_*)||\le \mu ||\mathcal{L}(\theta_t) - \mathcal{L}(\theta_*)||∣∣L(θt+1​)−L(θ∗​)∣∣≤μ∣∣L(θt​)−L(θ∗​)∣∣
where μ∈[0,1]\mu\in[0,1]μ∈[0,1]﻿ is the convergence rate.
For simple problems like the quadratic objective function, we can derive μ\muμ﻿ explicitly. If we use the steepest gradient descent with line search, the convergence rate is given by:
μ=(λmax⁡−λmin⁡λmax⁡+λmin⁡)2\mu=\Big(\frac{\lambda_{\max}-\lambda_{\min}}{\lambda_{\max}+\lambda_{\min}}\Big)^2μ=(λmax​+λmin​λmax​−λmin​​)2
Intuitively, the condition problem κ=λmax⁡λmin⁡\kappa=\frac{\lambda_{\max}}{\lambda_{\min}}κ=λmin​λmax​​﻿ represents the skewness of the space —i.e. being far from a symmetrical bowl.
More generally, the objective around the optimum is locally quadratic, hence the convergence rate depends on the condition number of the Hessian at that point: κ(H)\kappa(H)κ(H)﻿. We can improve the speed of convergence by using a surrogate model that has a Hessian close to the Hessian of the objective.
Line search often exhibits zig-zag paths that are inefficient, instead, we can use conjugate gradient descent.
