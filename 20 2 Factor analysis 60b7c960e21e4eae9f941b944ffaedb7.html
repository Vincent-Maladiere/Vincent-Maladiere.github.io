20.2 Factor analysis

PCA is a simple form of linear low-dimensional representation of the data.

Factor analysis is a generalization of PCA based on probabilistic model, so that we can treat it as building block for more complex models such as mixture of FA models or nonlinear FA models.

20.2.4 Unidentifiability of the parameters

The parameters of a FA model are unidentifiable. To see this, consider a model with weight W~=WR\tilde{W}=WRW~=WR﻿, where RRR﻿ is an arbitrary orthogonal rotation matrix, satisfying RR⊤=IRR^\top =IRR⊤=I﻿.

This has the same likelihood as model with weight WWW﻿, since:

Geometrically, multiplying z\bold{z}z﻿ by an orthogonal matrix is like rotating z\bold{z}z﻿ before generating x\bold{x}x﻿; but since z\bold{z}z﻿ is drawn from an anisotropic Gaussian distribution, it makes no difference on the likelihood.

Consequently, we can’t uniquely identify WWW﻿ and the latent factors either.

To break this symmetry, several solutions have been proposed:

  • Forcing WWW﻿ to have orthonormal columns. This is the approach adopted by PCA.
  • Forcing WWW﻿ to be lower triangular.
  • Sparsity promoting priors on the weight, by using ℓ1\ell_1ℓ1​﻿ regularization on WWW﻿.
  • Non gaussian prior p(z)p(\bold{z})p(z)﻿.
