7.4.7 Deflation

Suppose we have computed λ1\lambda_1λ1​﻿ using the power method, we now compute the subsequent λi\lambda_iλi​﻿.
We project out the u1\bold{u}_1u1​﻿ components from AAA﻿:
A(2)=(I−u1u1⊤)A(1)=A(1)−λ1u1u1⊤A^{(2)}=(I-\bold{u}_1\bold{u}_1^\top)A^{(1)}=A^{(1)}-\lambda_1 \bold{u}_1\bold{u}_1^\topA(2)=(I−u1​u1⊤​)A(1)=A(1)−λ1​u1​u1⊤​
This is called matrix deflation. We can now run the power method on A(2)A^{(2)}A(2)﻿ to find λ2\lambda_2λ2​﻿, the largest eigenvalue on the subspace orthogonal to u1\bold{u}_1u1​﻿.
Deflation can be used to implement PCA, by computing the first KKK﻿ eigenvectors of a covariance matrix.

7.4.8 Eigenvectors optimize quadratic forms

Consider the following constrained optimization problem:
max⁡x∈Rn  x⊤Ax,      subject  to  ∣∣x∣∣22=1\max_{x\in\mathbb{R}^n}\;\bold{x}^\top A\bold{x}, \;\;\;\mathrm{subject\;to\;}||\bold{x}||^2_2=1x∈Rnmax​x⊤Ax,subjectto∣∣x∣∣22​=1
for a symmetric matrix A∈SnA\in\mathbb{S}^nA∈Sn﻿.
A standard way to solve a constrained optimization problem is by forming the Lagrangian:
L(x,λ)=x⊤Ax+λ(1−x⊤x)\mathcal{L}(\bold{x},\lambda)=\bold{x}^\top A\bold{x}+\lambda(1-\bold{x}^\top \bold{x})L(x,λ)=x⊤Ax+λ(1−x⊤x)
Therefore:
∇xL(x,λ)=2A⊤x−2λx=0\nabla_x\mathcal{L}(\bold{x},\lambda)=2A^\top\bold{x}-2\lambda \bold{x}=0∇x​L(x,λ)=2A⊤x−2λx=0
So the problem is reduced to Ax=λxA\bold{x}=\lambda \bold{x}Ax=λx﻿, and the only solutions to maximize x⊤Ax\bold{x}^\top A\bold{x}x⊤Ax﻿ given x⊤x=1\bold{x}^\top\bold{x}=1x⊤x=1﻿ are the eigenvectors of AAA﻿.
