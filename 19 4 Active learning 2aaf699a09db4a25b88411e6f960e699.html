19.4 Active learning

In active learning, the goal is to identify the true predictive mapping y=f(x)y=f(\bold{x})y=f(x)﻿ by querying as few (x,y)(\bold{x},y)(x,y)﻿ points as possible. There are three variants.

In query synthesis, the algorithm gets to choose any input x\bold{x}x﻿ and can ask for its corresponding output y=f(x)y=f(\bold{x})y=f(x)﻿.

In pool-based active learning, there is large, but fixed, set of unlabeled points and the algorithm gets to ask a label for one or more of these points

Finally, in stream-based active learning the incoming data is arriving continuously, and the algorithm must choose whether it wants to request a label for the current input or not.

They are various closely related problems. In Bayesian optimization the goal is to estimate the location of the global optimum x∗=arg min⁡xf(x)\bold{x}^*=\argmin_{\bold{x}}f(\bold{x})x∗=argminx​f(x)﻿ in as few queries as possible. Typically, we fit a surrogate (response surface) model to the intermediate (x,y)(\bold{x},y)(x,y)﻿ queries, to decide which question to ask next.

In experiment design, the goal is to estimate p(θ∣D)p(\theta|\mathcal{D})p(θ∣D)﻿, using as little data as possible (this can thought of as an unsupervised form of active learning).
