11.5.1 B-spline basis functions

A spline is a piecewise polynomial of degree DDD﻿, where the locations of the pieces are defined by a set of knots t1<⋯<tmt_1<\dots<t_mt1​<⋯<tm​﻿.

The polynomial is defined on each of the intervals (−∞,t1],…,[tm,+∞](-\infin,t_1],\dots,[t_m,+\infin](−∞,t1​],…,[tm​,+∞]﻿. The function has continuous derivatives of order D−1D-1D−1﻿ at its knot points.

It is common to use cubic splines, where D=3D=3D=3﻿, ensuring the function has first and second-order derivatives at each knot.

We don’t focus on the inner working of the splines, it suffices to say we can use scikit-learn to convert the X∈RN×1X\in\mathbb{R}^{N\times 1}X∈RN×1﻿ matrix to B∈RN×(K+D+1)B\in\mathbb{R}^{N\times (K+D+1)}B∈RN×(K+D+1)﻿, with KKK﻿ the number of knots.

We see that for any given input, only D+1D+1D+1﻿ basis functions are active (non-zero). For the piecewise constant function, D=0D=0D=0﻿, for linear D=1D=1D=1﻿, and for cubic D=3D=3D=3﻿.

11.5.3 Smoothing splines

Smoothing splines are related to regression splines, but use NNN﻿ knots, the number of datapoints. They are non-parametric methods since the number of parameters is not fixed a priori but grows with the size of the data.

To avoid overfitting, smoothing splines rely on ℓ2\ell_2ℓ2​﻿ regularization.
