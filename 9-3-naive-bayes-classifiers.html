9.3 Naive Bayes classifiers

We discuss a simple generative approach where all features are considered conditionally independent given the class label.
Even if this assumption is not valid, it results in good classifiers. Since the model has only O(CD)O(CD)O(CD)﻿ features, it is immune to overfitting.
We use the following class conditional density:
p(x∣y=c,θ)=∏d=1Dp(xd∣y=c,θcd)p(x|y=c,\theta)=\prod_{d=1}^D p(x_d|y=c,\theta_{cd})p(x∣y=c,θ)=d=1∏D​p(xd​∣y=c,θcd​)
Hence, the posterior is:
p(y=c∣x,θ)=p(y=c∣π)∏d=1Dp(xd∣y=c,θcd)∑c′p(y=c′∣π)∏d=1Dp(xd∣y=c′,θc′d)p(y=c|x,\theta)=\frac{p(y=c|\pi)\prod_{d=1}^D p(x_d|y=c,\theta_{cd})}{\sum_{c'} p(y=c'|\pi)\prod_{d=1}^D p(x_d|y=c',\theta_{c'd})}p(y=c∣x,θ)=∑c′​p(y=c′∣π)∏d=1D​p(xd​∣y=c′,θc′d​)p(y=c∣π)∏d=1D​p(xd​∣y=c,θcd​)​
where πc\pi_cπc​﻿ is the prior probability density, and θ=(π,{θdc})\theta=(\pi,\{\theta_{dc}\})θ=(π,{θdc​})﻿ are all the parameters.

9.3.1 Example models

We need to specify the conditional likelihood, given the nature of the features:
If xdx_dxd​﻿ is binary, we can use the Bernoulli distribution:
p(x∣y=c,θ)=∏d=1DBer(xd∣θdc)p(x|y=c,\theta)=\prod_{d=1}^D Ber(x_d|\theta_{dc})p(x∣y=c,θ)=d=1∏D​Ber(xd​∣θdc​)
where θdc\theta_{dc}θdc​﻿ is the probability that xd=1x_d=1xd​=1﻿ in class ccc﻿.
This is called the Bernoulli multivariate naive model. This approach to MNIST has a surprisingly good 84% accuracy.
If xdx_dxd​﻿ is categorical, we can use the categorical distribution:
p(x∣y=c,θ)=∏d=1DCat(xd∣θcd)p(x|y=c,\theta)=\prod_{d=1}^D Cat(x_d|\theta_{cd})p(x∣y=c,θ)=d=1∏D​Cat(xd​∣θcd​)
If xd∈Rx_d \in \mathbb{R}xd​∈R﻿, we use the Gaussian distribution:
p(x∣y=c,θ)=∏d=1DN(xd∣μdc,σdc2)p(x|y=c,\theta)=\prod_{d=1}^D \mathcal{N}(x_d|\mu_{dc},\sigma^2_{dc})p(x∣y=c,θ)=d=1∏D​N(xd​∣μdc​,σdc2​)

9.3.3 Bayesian Naive Bayes

We now compute the posterior distribution over the parameters. Let's assume we have categorical features, so:
p(xd∣θdc)=Cat(xd∣θdc)p(x_d|\theta_{dc})=Cat(x_d|\theta_{dc})p(xd​∣θdc​)=Cat(xd​∣θdc​)
where θdck=p(xd=k∣y=c)\theta_{dck}=p(x_d=k|y=c)θdck​=p(xd​=k∣y=c)﻿.
We show in section 4.6.3 that the conjugate is the Dirichlet distribution:
p(θdc)=Dir(θdc∣βdc)p(\theta_{dc})=Dir(\theta_{dc}|\beta_{dc})p(θdc​)=Dir(θdc​∣βdc​)
Similarly, we use a Dirichlet distribution for the prior:
p(π)=Dir(π∣α)p(\pi)=Dir(\pi|\alpha)p(π)=Dir(π∣α)
We can compute the posterior in close form:
p(θ∣D)=Dir(π∣α˘)∏d=1D∏c=1CDir(θdc∣β˘dc)p(\theta|\mathcal{D})=Dir(\pi|\breve{\alpha}) \prod_{d=1}^D\prod_{c=1}^C Dir(\theta_{dc}|\breve{\beta}_{dc})p(θ∣D)=Dir(π∣α˘)d=1∏D​c=1∏C​Dir(θdc​∣β˘​dc​)
Where α˘c=αc+Nc\breve{\alpha}_c=\alpha_c+N_cα˘c​=αc​+Nc​﻿ and β˘dck=βdck+Ndck\breve{\beta}_{dck}=\beta_{dck}+N_{dck}β˘​dck​=βdck​+Ndck​﻿​
We derive the posterior predictive distribution as follows.
The prior over the label is given by:
p(y∣D)=Cat(y∣πˉ)p(y|\mathcal{D})=Cat(y|\bar{\pi})p(y∣D)=Cat(y∣πˉ)
where πˉc=αc˘∑c′α˘c′\bar{\pi}_c=\frac{\breve{\alpha_c}}{\sum_{c'} \breve{\alpha}_{c'}}πˉc​=∑c′​α˘c′​αc​˘​​﻿​
For the features, we have:
p(xd=k∣y=c,D)=θˉdckp(x_d=k|y=c,\mathcal{D})=\bar{\theta}_{dck}p(xd​=k∣y=c,D)=θˉdck​
where θˉdck=β˘dck∑k′β˘dck′=βdck+Nnck∑k′βdck′+Ndck′\bar{\theta}_{dck} = \frac{\breve{\beta}_{dck}}{\sum_{k'}\breve{\beta}}_{dck'} = \frac{\beta_{dck}+N_{nck}}{\sum_{k'}\beta_{dck'}+N_{dck'}}θˉdck​=∑k′​β˘​β˘​dck​​dck′​=∑k′​βdck′​+Ndck′​βdck​+Nnck​​﻿​
is the posterior mean of the parameters.
If βdck=0\beta_{dck}=0βdck​=0﻿, this reduces to the MLE
If βdck=1\beta_{dck}=1βdck​=1﻿, we add 1 to the empirical count before normalizing. This is called Laplace smoothing.
In the binary, case this gives us:
θˉdc=1+Ndc12+Ndc\bar{\theta}_{dc}=\frac{1+N_{dc1}}{2+N_{dc}}θˉdc​=2+Ndc​1+Ndc1​​
Once we have estimated the parameter posterior, the predicted distribution over the label is:
p(y=c∣x,D)∝p(y=c∣D)∏d=1Dp(xd∣y=c,D)=πˉc∏d∏kθˉdckI(xd=k)p(y=c|x,\mathcal{D}) \propto p(y=c|\mathcal{D})\prod_{d=1}^D p(x_d|y=c,\mathcal{D})=\bar{\pi}_c \prod_{d} \prod_k \bar{\theta}_{dck}^{\mathbb{I}(x_d=k)}p(y=c∣x,D)∝p(y=c∣D)d=1∏D​p(xd​∣y=c,D)=πˉc​d∏​k∏​θˉdckI(xd​=k)​
This gives us a fully Bayesian form of naive Bayes, in which we have integrated out all the parameters (the posterior mean parameters)

Let xdk=I(xd=k)x_{dk}=\mathbb{I}(x_d=k)xdk​=I(xd​=k)﻿, so xdx_{d}xd​﻿ is a one-hot encoding of feature ddd﻿.
