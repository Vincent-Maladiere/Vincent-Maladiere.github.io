21.4 Clustering using mixture models

We have seen how the K-means algorithm can be used to cluster data vectors in RD\R^DRD﻿. However, this method assumes that all clusters have the same spherical shape, which is a very restrictive assumption.
In addition, K-means assumes that all clusters can be described by Gaussian in the input space, so it can’t be applied to categorical data.
We can overcome both of these issues with mixture models.

21.4.2 Mixtures of Bernoullis

21.4.1 Mixtures of Gaussians

Recall from section 3.5.1 that a Gaussian mixture model (GMM) has the form:
p(x∣θ)=∑k=1KπkN(x∣μk,Σk)p(\bold{x}|\theta)=\sum_{k=1}^K \pi_k \mathcal{N}(\bold{x}|\mu_k,\Sigma_k)p(x∣θ)=k=1∑K​πk​N(x∣μk​,Σk​)
If we know the model parameters, θ=(π,{μk,Σk})\theta=(\pi,\{\mu_k,\Sigma_k\})θ=(π,{μk​,Σk​})﻿, we can use the Bayes rule to compute the responsibility (posterior membership probability) of a cluster kkk﻿ for the data point xn\bold{x}_nxn​﻿:
rnk≜p(zn=k∣xn,θ)=p(zn=k∣θ)p(xn∣zn=k,θ)∑k′p(zn=k′∣θ)p(xn∣zn=k′,θ)r_{nk}\triangleq p(z_n=k|\bold{x}_n,\theta)=\frac{p(z_n=k|\theta)p(\bold{x}_n|z_n=k,\theta)}{\sum_{k'}p(z_n=k'|\theta)p(\bold{x}_n|z_n=k',\theta)}rnk​≜p(zn​=k∣xn​,θ)=∑k′​p(zn​=k′∣θ)p(xn​∣zn​=k′,θ)p(zn​=k∣θ)p(xn​∣zn​=k,θ)​
We can then compute the most probable cluster assignment:
z^n=arg max⁡krnk=arg max⁡k[log⁡p(zn=k∣θ)+log⁡p(xn∣zn=k,θ)]\hat{z}_n=\argmax_k r_{nk}=\argmax_k [\log p(z_n=k|\theta) +\log p(\bold{x}_n|z_n=k,\theta)]z^n​=kargmax​rnk​=kargmax​[logp(zn​=k∣θ)+logp(xn​∣zn​=k,θ)]
This is known as hard clustering.
21.4.1.1 K-means is a special case of EM
We can estimate the parameter of a GMM using the EM algorithm (section 8.7.3).
It turns out K-means is a special case of this algorithm, which makes two assumptions:
We fix Σk=I\Sigma_k=IΣk​=I﻿ and πk=1/K\pi_k=1/Kπk​=1/K﻿ for all clusters, so we just have to estimate the means μk\mu_kμk​﻿​
We approximate the E step by replacing the soft responsibilities with hard assignment, i.e. we set rnk=I(k=zn∗)r_{nk}=\mathbb{I}(k=z_n^*)rnk​=I(k=zn∗​)﻿ instead of rnk=p(zn=k∣xn,θ)r_{nk}=p(z_n=k|\bold{x}_n,\theta)rnk​=p(zn​=k∣xn​,θ)﻿.
With this approximation, the weighted MLE problem of the M step reduces to equation:
μk=1Nk∑n:zn=kxn\mu_k=\frac{1}{N_k}\sum_{n:z_n=k}\bold{x}_nμk​=Nk​1​n:zn​=k∑​xn​
So we recover K-means.

21.4.2 Mixtures of Bernoullis

We can use a mixtures of Bernoulli to cluster binary data (as discussed in section 3.5.2). The model has the form:

Here μdk\mu_{dk}μdk​﻿ is the probability that bits ddd﻿ turns on in cluster kkk﻿.
We can fit this model using EM, SGD or MCMC.
