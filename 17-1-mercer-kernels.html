17.1 Mercer Kernels

A Mercer kernel or positive definite kernel is a symmetric function K:X×X→R+\mathcal{K}:\mathcal{X}\times \mathcal{X} \rightarrow \R_+K:X×X→R+​﻿​
Given a set of NNN﻿ data points, we define the Gram matrix as the following N×NN\times N N×N﻿ similarity matrix:
K=[K(x1,x1)…K(x1,xn)⋮⋱⋮K(xn,x1)…K(xn,xn)]K= \begin{bmatrix} \mathcal{K}(\bold{x}_1 ,\bold{x}_1)&\dots&\mathcal{K}(\bold{x}_1,\bold{x}_n) \\ \vdots & \ddots & \vdots \\ \mathcal{K}(\bold{x}_n ,\bold{x}_1) & \dots & \mathcal{K}(\bold{x}_n,\bold{x}_n) \end{bmatrix}K=⎣⎡​K(x1​,x1​)⋮K(xn​,x1​)​…⋱…​K(x1​,xn​)⋮K(xn​,xn​)​⎦⎤​
K\mathcal{K}K﻿ is a Mercer kernel iff the Gram matrix is positive definite for any set of distinct input {xn}n=1N\{x_n\}_{n=1}^N{xn​}n=1N​﻿​
The most widely used kernel is the RBF kernel:
K(x,x′)=exp⁡(−∣∣x−x′∣∣22ℓ2)\mathcal{K}(\bold{x},\bold{x}')=\exp\Big(-\frac{||\bold{x-x'}||^2}{2\ell^2}\Big)K(x,x′)=exp(−2ℓ2∣∣x−x′∣∣2​)
Here ℓ\ell ℓ﻿ corresponds is the bandwidth parameter i.e. the distance over which we expect differences to matter.
The RBF measures similarity between two vectors in RD\R^DRD﻿ using scaled Euclidean distance.

17.1.1 Mercer’s theorem

Any positive definite matrix KKK﻿ can be represented using an eigendecomposition of the form K=U⊤ΛUK=U^\top \Lambda UK=U⊤ΛU﻿ where Λ\LambdaΛ﻿ is the diagonal matrix of eigenvalues λi>0\lambda_i >0λi​>0﻿ and UUU﻿ is a matrix containing the eigenvectors.
Now consider the element (i,j)(i,j)(i,j)﻿ of KKK﻿:
kij=(Λ12U:i)⊤(Λ12U:j)k_{ij}=(\Lambda^{\frac{1}{2}}U_{:i})^\top(\Lambda^{\frac{1}{2}}U_{:j})kij​=(Λ21​U:i​)⊤(Λ21​U:j​)
where U:iU_{:i}U:i​﻿ is the iii﻿th column of UUU﻿.
If we define ϕ(xi)=Λ12U:i\phi(\bold{x}_i)=\Lambda^\frac{1}{2} U_{:i}ϕ(xi​)=Λ21​U:i​﻿ we can write:
kij=ϕ(xi)⊤ϕ(xj)=∑mϕm(xi)⊤ϕm(xj)k_{ij}=\phi(\bold{x}_i)^\top \phi(\bold{x}_j)=\sum_m \phi_m(\bold{x}_i)^\top \phi_m(\bold{x}_j)kij​=ϕ(xi​)⊤ϕ(xj​)=m∑​ϕm​(xi​)⊤ϕm​(xj​)
The entries of the kernel matrix can be computed by performing an inner product of some feature vectors that are implicitly defined by the eigenvectors of the kernel matrix.
This idea can be generalized to apply to kernel function, not just kernel matrices. For example, consider the quadratic kernel K(x,x′)=⟨x,x′⟩2\mathcal{K}(\bold{x},\bold{x'})=\lang \bold{x},\bold{x}'\rang^2K(x,x′)=⟨x,x′⟩2﻿. In 2d, we have:
K(x,x′)=(x1x1′+x2x2′)2=x12(x1′)2+2x1x1′x2x2′+x22(x2′)2\mathcal{K}(\bold{x},\bold{x}')=(x_1x_1'+x_2x_2')^2=x_1^2(x_1')^2+2 x_1x_1'x_2x_2'+x_2^2(x_2')^2K(x,x′)=(x1​x1′​+x2​x2′​)2=x12​(x1′​)2+2x1​x1′​x2​x2′​+x22​(x2′​)2
We can write this:
K(x,x′)=ϕ(x)⊤ϕ(x′)\mathcal{K}(\bold{x},\bold{x}')=\phi(\bold{x})^\top \phi(\bold{x}')K(x,x′)=ϕ(x)⊤ϕ(x′)
where ϕ(x)=[x12,2x1x2,x22]∈R3\phi(\bold{x})=[x_1^2,\sqrt{2}x_1x_2,x_2^2]\in \R^3ϕ(x)=[x12​,2​x1​x2​,x22​]∈R3﻿.
So, we embedded the 2d input x\bold{x}x﻿ into a 3d feature space ϕ(x)\phi(\bold{x})ϕ(x)﻿.
The feature representation of the RBF kernel is infinite dimensional. However, by working with kernel functions, we can avoid having to deal with infinite dimensional vectors.

17.1.2 Popular Mercer kernels

17.1.2.1 Stationary kernels for real-valued vectors
For real-valued inputs, X=RD\mathcal{X}=\R^DX=RD﻿, it is common to use stationary kernels, of the form:
K(x,x′)=K(∣∣x−x′∣∣)\mathcal{K}(\bold{x},\bold{x}')=\mathcal{K}(||\bold{x}-\bold{x}'||)K(x,x′)=K(∣∣x−x′∣∣)
The value of the output only depends on the distance between the inputs. The RBF kernel is a stationary kernel. We give some examples below.
ARD Kernel
We can generalize the RBF kernel by replacing the Euclidean distance with the Mahalanobis distance:
K(r)=σ2exp⁡(−12r⊤Σ−1r)\mathcal{K}(\bold{r})=\sigma^2\exp\Big(-\frac{1}{2}\bold{r}^\top \Sigma^{-1} \bold{r}\Big)K(r)=σ2exp(−21​r⊤Σ−1r)
where r=x−x′\bold{r=x-x'}r=x−x′﻿.
If Σ\Sigma Σ﻿ is diagonal, this can be written:
K(r;ℓ,σ2)=σ2exp⁡(−12∑d=1Drd2ℓd2)=∏d=1DK(rd,ℓd,σ2/d)\mathcal{K}(\bold{r};\ell,\sigma^2)=\sigma^2\exp\Big(-\frac{1}{2}\sum_{d=1}^D \frac{r_d^2}{\ell_d^2} \Big)=\prod_{d=1}^D \mathcal{K}(r_d,\ell_d,\sigma^{2/d})K(r;ℓ,σ2)=σ2exp(−21​d=1∑D​ℓd2​rd2​​)=d=1∏D​K(rd​,ℓd​,σ2/d)
where:
K(r,ℓ,τ2)=τ2exp⁡(−12r2ℓ2)\mathcal{K}(r,\ell,\tau^2)=\tau^2\exp\Big(-\frac{1}{2}\frac{r^2}{\ell^2}\Big)K(r,ℓ,τ2)=τ2exp(−21​ℓ2r2​)
We can interpret σ2\sigma^2σ2﻿ as the total variance and ℓd\ell_dℓd​﻿ as defining the characteristic length scale of dimension ddd﻿. If ddd﻿ is an irrelevant input dimension, we can set ℓd=∞\ell_d=\infinℓd​=∞﻿, so the corresponding dimension will be ignore.
This is known as automatic relevancy determination (ARD), hence this kernel is called ARD kernel.
Matern kernels
The SE kernel yields functions that are infinitely differentiable, and therefore very smooth. For many applications, it is better to use the Matern kernel which yields rougher functions.
This allow to model small “wiggles” without having to make the overall length scale very small.
The Matern kernel has the following form:
K(r;v,ℓ)=21−νΓ(ν)(2νrℓ)νKν(2νrℓ)\mathcal{K}(r;v,\ell)=\frac{2^{1-\nu}}{\Gamma(\nu)}\Big(\frac{\sqrt{2}\nu r}{\ell}\Big)^\nu K_\nu\Big(\frac{\sqrt{2}\nu r}{\ell}\Big)K(r;v,ℓ)=Γ(ν)21−ν​(ℓ2​νr​)νKν​(ℓ2​νr​)
where KvK_vKv​﻿ is a modified Bessel function and ℓ\ell ℓ﻿ the length scale.
Functions sample from this GP are kkk﻿-times differentiable, where ν>k\nu>kν>k﻿. As ν→∞\nu\rightarrow \infinν→∞﻿, this approaches the SE kernel.
For value ν=12\nu =\frac{1}{2}ν=21​﻿, the function simplifies as follow:
K(r;12,ℓ)=exp⁡(−rℓ)\mathcal{K}(r;\frac{1}{2},\ell)=\exp(-\frac{r}{\ell})K(r;21​,ℓ)=exp(−ℓr​)
This corresponds to the Ornstein-Uhlenbeck process which describe the velocity of a particle undergoing Brownian motion. The corresponding function is continuous but very jagged.
Periodic kernels
The periodic kernel capture repeating structure, and has the form:
Kper(r;ℓ,p)=exp⁡(−2ℓ2sin⁡2(πrp))\mathcal{K}_{\mathrm{per}}(r;\ell,p)=\exp\Big(-\frac{2}{\ell^2}\sin^2(\pi\frac{r}{p})\Big)Kper​(r;ℓ,p)=exp(−ℓ22​sin2(πpr​))
where ppp﻿ is the period.
This is related to the cosine kernel:
K(r;p)=cos⁡(2πrp)\mathcal{K}(r;p)=\cos\Big(2\pi \frac{r}{p}\Big)K(r;p)=cos(2πpr​)
17.1.2.2 Making new kernels from old
Given a valid kernel K1(x,x′)\mathcal{K}_1(\bold{x,x}')K1​(x,x′)﻿, we can create new kernel using any of the following methods:

K(x,x′)=x⊤Ax\mathcal{K}(\bold{x,x}')=\bold{x}^\top A \bold{x}K(x,x′)=x⊤Ax﻿, for any psd matrix AAA﻿​
For example, suppose we start with the linear kernel K(x,x′)=x⊤x′\mathcal{K}(\bold{x,x}')=\bold{x}^\top \bold{x}'K(x,x′)=x⊤x′﻿.
We know this is a valid Mercer kernel since the Gram matrix is the (scaled) covariance matrix of the data.
From the above rules we see that the polynomial kernel K(x,x′)=(x⊤x)M\mathcal{K}(\bold{x,x}')=(\bold{x}^\top\bold{x})^MK(x,x′)=(x⊤x)M﻿ is also a valid kernel, and contains all monomial of order MMM﻿​
We can also infer that the Gaussian kernel is a valid kernel. We can see this by noting:
∣∣x−x′∣∣2=x⊤x+x′⊤x′−2x⊤x′||\bold{x-x}'||^2=\bold{x}^\top \bold{x}+\bold{x}'^\top \bold{x}'-2\bold{x}^\top \bold{x}'∣∣x−x′∣∣2=x⊤x+x′⊤x′−2x⊤x′

K(x,x′)=exp⁡(−∣∣x−x′∣∣2/2σ2)=exp⁡(−x⊤x/2σ2)exp⁡(x⊤x′/σ2)exp⁡(−x′⊤x′/2σ2)\begin{align} \mathcal{K}(\bold{x,x}')&=\exp(-||\bold{x-x}'||^2/2\sigma^2)\\ &=\exp(-\bold{x}^\top\bold{x}/2\sigma^2)\exp(\bold{x}^\top\bold{x}'/\sigma^2)\exp(-\bold{x}'^\top\bold{x}'/2\sigma^2) \end{align}K(x,x′)​=exp(−∣∣x−x′∣∣2/2σ2)=exp(−x⊤x/2σ2)exp(x⊤x′/σ2)exp(−x′⊤x′/2σ2)​​
We scale by a constant, then use the exponential, then symmetrically multiply by functions of x\bold{x}x﻿ and x′\bold{x}'x′﻿.
17.1.2.3 Combining kernels by addition and multiplication
We can also combine kernels using addition or multiplication:
K(x,x′)=K1(x,x′)+K2(x,x′)\mathcal{K}(\bold{x,x}')=\mathcal{K}_1(\bold{x,x}')+\mathcal{K}_2(\bold{x,x}')K(x,x′)=K1​(x,x′)+K2​(x,x′)﻿​
K(x,x′)=K1(x,x′)×K2(x,x′)\mathcal{K}(\bold{x,x}')=\mathcal{K}_1(\bold{x,x}')\times \mathcal{K}_2(\bold{x,x}')K(x,x′)=K1​(x,x′)×K2​(x,x′)﻿​
Adding (resp. multiplying) positive-definite kernels together always result in another positive definite kernel. This is a way to get a disjunction (resp. conjunction) of the individual properties of each kernel.
17.1.2.4 Kernels for structured inputs
Kernels are particularly useful for structured inputs, since it’s often hard to “featurize” variable-size inputs.
We can define a string kernel by comparing strings in terms of the number of ngrams they have in common.
We can also define kernel on graph. For example random walk kernel perform random walk simultaneously on two graphs and count the number of paths in common.
