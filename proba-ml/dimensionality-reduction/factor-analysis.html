20.2 Factor Analysis

20.2 Factor analysis

PCA is a simple form of linear low-dimensional representation of the data.

Factor analysis is a generalization of PCA based on probabilistic model, so that we can treat it as building block for more complex models such as mixture of FA models or nonlinear FA models.

20.2.1 Generative model

Factor analysis is a linear-Gaussian latent variable generative model:

p(z)=N(z∣μ0,Σ0)p(x∣z,θ)=N(x∣Wz+μ∣Ψ)\begin{align} p(\bold{z}) &=\mathcal{N}(\bold{z}|\mu_0,\Sigma_0)\\ p(\bold{x}|\bold{z},\theta)&=\mathcal{N}(\bold{x}|W\bold{z}+\mu|\Psi) \end{align} p(z)p(x∣z,θ)​=N(z∣μ0​,Σ0​)=N(x∣Wz+μ∣Ψ)​​

where W∈RD×LW\in\R^{D\times L}W∈RD×L is known as the factor loading matrix and Ψ∈RD×D\Psi\in\R^{D\times D}Ψ∈RD×D is the covariance matrix.

FA can be thought as a low-rank version of a Gaussian distribution. To see this, note that the following induced marginal distribution is Gaussian:

p(x∣θ)=∫p(x∣z,θ)p(z)dz=∫N(x∣Wz+μ,Ψ)N(z∣μ0,Σ0)dz=N(x∣Wμ0+μ,Ψ+WΣ0W⊤)\begin{align} p(\bold{x}|\theta)&=\int p(\bold{x}|\bold{z},\theta)p(\bold{z})d\bold{z} \\ &= \int \mathcal{N}(\bold{x}|W\bold{z}+\mu,\Psi)\mathcal{N}(\bold{z}|\mu_0,\Sigma_0)d\bold{z} \\ &=\mathcal{N}(\bold{x}|W\mu_0+\mu,\Psi + W \Sigma_0W^\top) \end{align}p(x∣θ)​=∫p(x∣z,θ)p(z)dz=∫N(x∣Wz+μ,Ψ)N(z∣μ0​,Σ0​)dz=N(x∣Wμ0​+μ,Ψ+WΣ0​W⊤)​​

Without loss of generality, we set μ0=0\mu_0=\bold{0}μ0​=0 and Σ0=I\Sigma_0=IΣ0​=I.

We get:

p(z)=N(z∣0,I)p(x∣z)=N(x∣Wz+μ,Ψ)p(x)=N(x∣μ,Ψ+WW⊤)\begin{align} p(\bold{z})&=\mathcal{N}(\bold{z}|\bold{0},I)\\ p(\bold{x}|\bold{z})&=\mathcal{N}(\bold{x}|W\bold{z}+\mu,\Psi)\\ p(\bold{x})&= \mathcal{N}(\bold{x}|\mu,\Psi+WW^\top) \end{align}p(z)p(x∣z)p(x)​=N(z∣0,I)=N(x∣Wz+μ,Ψ)=N(x∣μ,Ψ+WW⊤)​​

Suppose we use L=1L=1L=1 and D=2D=2D=2 and Ψ=σ2I\Psi=\sigma^2IΨ=σ2I.

We can see it as taking an isotropic Gaussian “spray can” representing the likelihood p(x∣z)p(\bold{x|z})p(x∣z), and sliding it along the 1d line defined by zw+μz\bold{w}+\muzw+μ as we vary the 1d latent prior p(z)p(z)p(z).

In general, FA approximates the low rank decomposition of the visible covariance matrix:

C=Cov[x]=WW⊤+ΨC=\mathrm{Cov}[\bold{x}]=WW^\top +\PsiC=Cov[x]=WW⊤+Ψ

This only uses O(LD)O(LD)O(LD) parameters, which allows a flexible compromise between a full covariance Gaussian with O(D2)O(D^2)O(D2) parameters, and a diagonal covariance with O(D)O(D)O(D) parameters.

From this equation, we see that we should restrict Ψ\PsiΨ to be diagonal, otherwise we could set W=0W=\bold{0}W=0, ignoring latent factors while still being able to model any covariance.

The marginal variance of each visible variable is:

V[xd]=∑k=1Lwdk2+ψd\mathbb{V}[x_d]=\sum_{k=1}^L w_{dk}^2+\psi_dV[xd​]=k=1∑L​wdk2​+ψd​

where the first term is the variance due to the common factors, and the uniqueness ψd\psi_dψd​ is the variance specific to that dimension.

We can estimate the parameters of a FA using EM. Once we have fit the model, we can compute probabilistic latent embeddings using p(z∣x)p(\bold{z}|\bold{x})p(z∣x).

Using Bayes rule for Gaussians, we have:

p(z∣x)=N(z∣W⊤C−1(y−μ),I−W⊤C−1W)p(\bold{z}|\bold{x})=\mathcal{N}(\bold{z}|W^\top C^{-1}(\bold{y}-\mu),I-W^\top C^{-1}W)p(z∣x)=N(z∣W⊤C−1(y−μ),I−W⊤C−1W)

20.2.2 Probabilistic PCA

We consider a special case when of the factor analysis in which WWW has orthonormal columns, Ψ=σ2I\Psi=\sigma^2IΨ=σ2I \Psi=\sigma^2I$$\mu=\bold{0}. This is called probabilistic PCA or sensible PCA.

The marginal distribution on the visible variables has the form:

p(x∣θ)=∫N(x∣μ,σ2I)N(z∣0,I)dz=N(x∣μ,C)p(\bold{x}|\theta)=\int \mathcal{N}(\bold{x}|\mu,\sigma^2I)\mathcal{N}(\bold{z}|\bold{0},I)d\bold{z}=\mathcal{N}(\bold{x}|\mu ,C)p(x∣θ)=∫N(x∣μ,σ2I)N(z∣0,I)dz=N(x∣μ,C)

where:

C=σ2I+WW⊤C=\sigma^2I+WW^\top C=σ2I+WW⊤

The log likelihood is:

log⁡p(x∣W,μ,σ2)=−N2[Dlog⁡2π+log⁡∣C∣]−12∑n=1N(xn−μ)⊤C−1(xn−μ)=−N2[Dlog⁡2π+log⁡∣C∣+tr(C−1S)]\begin{align} \log p(\bold{x}|W,\mu,\sigma^2)&=-\frac{N}{2}[D\log 2\pi+\log|C|]-\frac{1}{2}\sum_{n=1}^N(\bold{x}_n-\mu)^\top C^{-1}(\bold{x}_n-\mu) \\ &= -\frac{N}{2}[D\log 2\pi+\log |C|+\mathrm{tr}(C^{-1}S)] \end{align}logp(x∣W,μ,σ2)​=−2N​[Dlog2π+log∣C∣]−21​n=1∑N​(xn​−μ)⊤C−1(xn​−μ)=−2N​[Dlog2π+log∣C∣+tr(C−1S)]​​

where we plugged xˉ\bar{\bold{x}}xˉ, the MLE of μ\muμ, and used the trace trick on the empirical covariance matrix S=1N∑n(xn−xˉ)(xn−xˉ)⊤S=\frac{1}{N}\sum_{n}(\bold{x}_n-\bar{\bold{x}})(\bold{x}_n-\bar{\bold{x}})^\topS=N1​∑n​(xn​−xˉ)(xn​−xˉ)⊤

It has been shown that the maximum of this objective must satisfy:

W=UL(LL−σ2I)1/2W=U_L(L_L-\sigma^2I)^{1/2}W=UL​(LL​−σ2I)1/2

where UL∈RD×LU_L\in\R^{D\times L}UL​∈RD×L are the eigenvectors associated to the LLL largest eigenvalues of SSS, and LL∈RL×LL_L\in \R^{L\times L}LL​∈RL×L is the diagonal eigenvalues matrix.

In the noise free limit, where σ2=0\sigma^2=0σ2=0, we get:

Wmle=ULLL1/2W_{\mathrm{mle}}=U_L L_L^{1/2}Wmle​=UL​LL1/2​

which is proportional to the PCA solution.

The MLE for the observation variance is:

σ2=1D−L∑i=L+1Dλi\sigma^2=\frac{1}{D-L}\sum_{i=L+1}^D \lambda_iσ2=D−L1​i=L+1∑D​λi​

which is the average distortion associated with the discarded dimensions.

20.2.4 Unidentifiability of the parameters

The parameters of a FA model are unidentifiable. To see this, consider a model with weight W~=WR\tilde{W}=WRW~=WR, where RRR is an arbitrary orthogonal rotation matrix, satisfying RR⊤=IRR^\top =IRR⊤=I.

This has the same likelihood as model with weight WWW, since:

Cov[x]=W~E[zz⊤]W~⊤+E[xx⊤]=WRR⊤W⊤+Ψ=WW⊤+Ψ\begin{align} \mathrm{Cov}[\bold{x}]&= \tilde{W}\mathbb{E}[\bold{z}\bold{z}^\top]\tilde{W}^\top+\mathbb{E}[\bold{x}\bold{x}^\top]\\ &= WRR^\top W^\top +\Psi \\ &=WW^\top +\Psi \end{align}Cov[x]​=W~E[zz⊤]W~⊤+E[xx⊤]=WRR⊤W⊤+Ψ=WW⊤+Ψ​​

Geometrically, multiplying z\bold{z}z by an orthogonal matrix is like rotating z\bold{z}z before generating x\bold{x}x; but since z\bold{z}z is drawn from an anisotropic Gaussian distribution, it makes no difference on the likelihood.

Consequently, we can’t uniquely identify WWW and the latent factors either.

To break this symmetry, several solutions have been proposed:

  • Forcing WWW to have orthonormal columns. This is the approach adopted by PCA.
  • Forcing WWW to be lower triangular.
  • Sparsity promoting priors on the weight, by using ℓ1\ell_1ℓ1​ regularization on WWW.
  • Non gaussian prior p(z)p(\bold{z})p(z).

20.2.5 Nonlinear factor analysis

The FA model assumes the observed data can be modeled with a linear mapping from a low-dimensional set of Gaussian factors.

One way to relax such assumption is to let the mapping from z\bold{z}z to x\bold{x}x be a nonlinear model, such as a neural network. The model becomes:

p(x∣θ)=∫N(x∣f(z;θ),Ψ)N(z∣0,I)dzp(\bold{x}|\theta)=\int \mathcal{N}(\bold{x}|f(\bold{z};\theta),\Psi)\mathcal{N}(\bold{z}|\bold{0},I)d\bold{z}p(x∣θ)=∫N(x∣f(z;θ),Ψ)N(z∣0,I)dz

This is called nonlinear factor analysis.

Unfortunately, we can’t compute the posterior or MLE exactly. Variational autoencoders is the most common way to approximate nonlinear FA model.

20.2.6 Mixture of factor analysers

One other way to relax the assumption made by FA is assuming the model is only locally linear, so the overall model becomes a weighted combination of FA models. This is called a mixture of FA (MFA).

The overall model for the data is a mixture of linear manifolds, which can be used to approximate the overall curved manifold.

Let cn∈{1,…,K}c_n\in\{1,\dots,K\}cn​∈{1,…,K} be the latent indicator which specify which subspace we should use to generate the data.

If cn=kc_n=kcn​=k, we sample zn\bold{z}_nzn​ from a Gaussian prior and pass it through the WkW_kWk​ matrix and add noise.

p(z∣θ)=N(z∣0,I)p(cn∣θ)=Cat(cn∣π)p(xn∣zn,cn=k,θ)=N(xn∣Wkzn+μk,Ψk)\begin{align} p(\bold{z}|\theta)&=\mathcal{N}(\bold{z}|\bold{0},I)\\ p(c_n|\theta)&= \mathrm{Cat}(c_n|\pi) \\ p(\bold{x}_n|\bold{z}_n,c_n=k,\theta)&=\mathcal{N}(\bold{x}_n|W_k\bold{z}_n+\mu_k,\Psi_k) \end{align}p(z∣θ)p(cn​∣θ)p(xn​∣zn​,cn​=k,θ)​=N(z∣0,I)=Cat(cn​∣π)=N(xn​∣Wk​zn​+μk​,Ψk​)​​

The distribution in the visible space is:

p(x∣θ)=∑kp(c=k)∫p(x∣z,c)p(z∣c)dz=∑kπk∫N(x∣Wkz,σ2I)N(z∣μk,I)dz\begin{align} p(\bold{x}|\theta)&=\sum_k p(c=k)\int p(\bold{x}|\bold{z},c)p(\bold{z}|c)d\bold{z} \\ &= \sum_k \pi_k \int \mathcal{N}(\bold{x}|W_k\bold{z,}\sigma^2I)\mathcal{N}(\bold{z}|\mu_k,I)d\bold{z} \end{align}p(x∣θ)​=k∑​p(c=k)∫p(x∣z,c)p(z∣c)dz=k∑​πk​∫N(x∣Wk​z,σ2I)N(z∣μk​,I)dz​​

In the special case of Ψ=σ2I\Psi=\sigma^2IΨ=σ2I we get a mixture of PPCA (although it is diffucilt to ensure orthogonality of the WkW_kWk​ in this case).
