20. Dimensionality Reduction
20.1 Principal Component Analysis (PCA)

20.1 Principal component analysis (PCA)

The simplest and most widely used form of dimensionality reduction is PCA. The basic idea is to find a linear and orthogonal projection of the high dimensional data x∈RD\bold{x}\in \R^Dx∈RD to a low dimensional subspace z∈RL\bold{z}\in \R^Lz∈RL, such that the low dimensional representation is a “good approximation” to the original data.

More specifically, if we project or encode x\bold{x}x to get z=W⊤x\bold{z}=W^\top \bold{x}z=W⊤x and then unproject or decode z\bold{z}z to get x^=Wz\hat{\bold{x}}=W\bold{z}x^=Wz, then we want x\bold{x}x and x^\hat{\bold{x}}x^ to be close in the ℓ2\ell_2ℓ2​ distance.

We define the following reconstruction error or distortion:

L(W)≜1N∑n=1N∣∣xn−decode(encode(xn;W);W))∣∣22\mathcal{L}(W)\triangleq \frac{1}{N}\sum_{n=1}^N ||\bold{x}_n-\mathrm{decode}(\mathrm{encode(\bold{x}_n;W);W}))||_2^2L(W)≜N1​n=1∑N​∣∣xn​−decode(encode(xn​;W);W))∣∣22​

where the encoder and decoder stages are both linear maps.

We can minimize this objective by setting W^=UL\hat{W}=U_LW^=UL​, where ULU_LUL​ contains the LLL eigenvectors with the largest eigenvalues of the empirical covariance matrix:

Σ^=1N∑n=1N(xn−x^)(xn−x^)⊤=1NXc⊤Xc\hat{\Sigma}=\frac{1}{N}\sum_{n=1}^N (\bold{x}_n-\hat{\bold{x}})(\bold{x}_n-\hat{\bold{x}})^\top=\frac{1}{N}X_c^\top X_cΣ^=N1​n=1∑N​(xn​−x^)(xn​−x^)⊤=N1​Xc⊤​Xc​

where XcX_cXc​ is the centered version of the N×DN\times DN×D design matrix.

This is equivalent to maximizing the likelihood of a latent linear Gaussian model known as probabilistic PCA.

20.1.2 Derivation of the algorithm

Suppose we have an unlabeled dataset D={xn:n=1:N}\mathcal{D}=\{\bold{x}_n:n=1:N\}D={xn​:n=1:N} where xn∈RD\bold{x}_n\in \R^Dxn​∈RD. We can represent this as a X∈RN×DX\in \R^{N\times D}X∈RN×D data matrix.

We assume xˉ=1N∑n=1Nxn=0\bar{\bold{x}}=\frac{1}{N}\sum_{n=1}^N \bold{x}_n=\bold{0}xˉ=N1​∑n=1N​xn​=0, which can be assured by centering the data.

We would like to approximate each xn\bold{x}_nxn​ by a latent vector zn∈RL\bold{z}_n\in \R^Lzn​∈RL. The collection of these latent variable are called the latent factors.

We assume each xn\bold{x}_nxn​ can be “explained” in terms of a weighed combination of basis functions w1,…,wL\bold{w}_1,\dots,\bold{w}_Lw1​,…,wL​ where each wk∈RD\bold{w}_k\in \R^Dwk​∈RD and the weigths are zn∈RL\bold{z}_n\in \R^Lzn​∈RL, i.e we assume:

xn≈∑k=1Lzn,kwk\bold{x}_n\approx \sum_{k=1}^L z_{n,k}\bold{w}_kxn​≈k=1∑L​zn,k​wk​

We can measure the reconstruction error produced by this approximation as:

L(W,Z)=1N∣∣X−ZW⊤∣∣F2=1N∣∣X⊤−WZ⊤∣∣F2=1N∑n=1N∣∣xn−Wzn∣∣22\begin{align} \mathcal{L}(W,Z)&=\frac{1}{N}||X-ZW^\top||^2_F\\ &=\frac{1}{N}||X^\top-WZ^\top||^2_F \\ &=\frac{1}{N}\sum_{n=1}^N ||\bold{x}_n-W\bold{z}_n||^2_2 \end{align}L(W,Z)​=N1​∣∣X−ZW⊤∣∣F2​=N1​∣∣X⊤−WZ⊤∣∣F2​=N1​n=1∑N​∣∣xn​−Wzn​∣∣22​​​

We want to minimize this subject to the constraint that WWW is an orthogonal matrix.

We show below that the optimal solution is obtained by setting W^=UL\hat{W}=U_LW^=UL​ where ULU_LUL​ contains the LLL eigenvectors with largest eigenvalues of the empirical covariance matrix.

20.1.2.1 Base case

We start by estimating the best 1d solution w1∈RD\bold{w}_1\in \R^Dw1​∈RD. We will find the remaining basis vectors later.

Let the latent coefficient \bold{}for each data points associated with the first basis vector be z~1=[z1,1,…,zN,1]∈RN\tilde{\bold{z}}_1=[z_{1,1},\dots,z_{N,1}]\in \R^Nz~1​=[z1,1​,…,zN,1​]∈RN

The reconstruction error is given by:

L(w1,z~1)=1N∑n=1N∣∣xn−z1,nw1∣∣22=1N∑n=1N(xn−z1,nw1)⊤(xn−z1,nw1)=1N∑n=1Nxn⊤xn−2z1,nw1⊤xn+z1,n2w1⊤w1\begin{align} \mathcal{L}(\bold{w}_1,\tilde{\bold{z}}_1)&=\frac{1}{N}\sum_{n=1}^N ||\bold{x}_n-z_{1,n}\bold{w}_1||^2_2\\ &=\frac{1}{N}\sum_{n=1}^N (\bold{x}_n-z_{1,n}\bold{w}_1)^\top(\bold{x}_n-z_{1,n}\bold{w}_1) \\ &= \frac{1}{N} \sum_{n=1}^N \bold{x}_n^\top \bold{x}_ n-2z_{1,n}\bold{w}_1^\top\bold{x}_n+z_{1,n}^2\bold{w}_1^\top \bold{w}_1 \end{align}L(w1​,z~1​)​=N1​n=1∑N​∣∣xn​−z1,n​w1​∣∣22​=N1​n=1∑N​(xn​−z1,n​w1​)⊤(xn​−z1,n​w1​)=N1​n=1∑N​xn⊤​xn​−2z1,n​w1⊤​xn​+z1,n2​w1⊤​w1​​​

where w1⊤w1=1\bold{w}_1^\top \bold{w}_1=1w1⊤​w1​=1 by the orthonormality assumption.

Taking derivative wrt z1,nz_{1,n}z1,n​ gives:

∂∂z1,nL(w1,z~1)=−2w1xn+2z1,n=0  ⟹  z1,n=w1⊤xn\begin{align} \frac{\partial}{\partial z_{1,n}}\mathcal{L}(\bold{w}_1,\tilde{\bold{z}}_1)&=-2\bold{w}_1\bold{x}_n+2z_{1,n}=0 \\ &\implies z_{1,n}=\bold{w}_1^\top\bold{x}_n \end{align}∂z1,n​∂​L(w1​,z~1​)​=−2w1​xn​+2z1,n​=0⟹z1,n​=w1⊤​xn​​​

So the optimal embedding is obtained by orthogonally projecting the data onto w1\bold{w}_1w1​.

Plugging it back to loss gives us:

L(w1,z~1∗)=1N∑n=1Nxn⊤xn−z1,n2=C−1N∑n=1Nz1,n2=C−1N∑n=1Nw1⊤xnxn⊤w1=C−w1⊤Σ^w1\begin{align} \mathcal{L}(\bold{w}_1,\tilde{\bold{z}}_1^*)&=\frac{1}{N}\sum_{n=1}^N \bold{x}_n^\top\bold{x}_n -z_{1,n}^2\\ &=C-\frac{1}{N}\sum_{n=1}^N z_{1,n}^2 \\ &= C -\frac{1}{N}\sum_{n=1}^N \bold{w}_1^\top \bold{x}_n \bold{x}_n^\top \bold{w}_1\\ &=C-\bold{w}_1^\top \hat{\Sigma}\bold{w}_1 \end{align}L(w1​,z~1∗​)​=N1​n=1∑N​xn⊤​xn​−z1,n2​=C−N1​n=1∑N​z1,n2​=C−N1​n=1∑N​w1⊤​xn​xn⊤​w1​=C−w1⊤​Σ^w1​​​

Where Σ\SigmaΣ is the empirical covariance matrix (since we assumed the data is centered) and CCC a constant.

We can trivially optimize this by letting w1→∞\bold{w}_1\rightarrow \infinw1​→∞, so we impose the constraint ∣∣w1∣∣2=1||\bold{w}_1||_2=1∣∣w1​∣∣2​=1 and instead optimize:

L~(w1,z1∗)=−w1⊤Σw1+λ1(w1⊤w1−1)\begin{equation} \tilde{\mathcal{L}}(\bold{w}_1,\bold{z}_1^*)=-\bold{w}_1^\top\Sigma\bold{w}_1+\lambda_1(\bold{w}_1^\top \bold{w}_1-1) \end{equation}L~(w1​,z1∗​)=−w1⊤​Σw1​+λ1​(w1⊤​w1​−1)​​

where λ1\lambda_1λ1​ is a Lagrange multiplier.

Taking derivatives and equating to zero we have:

∂∂w1L(w1,z1∗)=−2w1Σ^+2λ1w1=0  ⟹  Σ^w1=λ1w1\begin{align} \frac{\partial }{\partial \bold{w}_1}\mathcal{L}(\bold{w}_1,\bold{z}_1^*)&=-2\bold{w_1}\hat{\Sigma}+2\lambda_1\bold{w}_1=0 \\ &\implies \hat{\Sigma }\bold{w}_1=\lambda_1 \bold{w}_1 \end{align}∂w1​∂​L(w1​,z1∗​)​=−2w1​Σ^+2λ1​w1​=0⟹Σ^w1​=λ1​w1​​​

Hence, the optimal direction onto which should project the data is an eigenvector of the covariance matrix.

And since we have:

w1⊤Σ^w1=λ1\bold{w}_1^\top \hat{\Sigma} \bold{w_1}=\lambda_1w1⊤​Σ^w1​=λ1​

We can minimize the loss in (13) by picking the largest eigenvector which corresponds to the largest eigenvalue.

20.1.2.2 Optimal weight vector maximizes the variance of the projected data

Before continuing, we observe that, since the data has been centered:

E[z1,n]=E[w1⊤xn]=w1⊤E[xn]=0\mathbb{E}[z_{1,n}]=\mathbb{E}[\bold{w}_1^\top\bold{x}_n]=\bold{w}_1^\top\mathbb{E}[\bold{x}_n]=0E[z1,n​]=E[w1⊤​xn​]=w1⊤​E[xn​]=0

Hence the variance of the projected data is given by:

V[z~1]=E[z~12]=1N∑n=1Nz1,n2=−L(w1)+C\mathbb{V}[\tilde{\bold{z}}_1]=\mathbb{E}[\tilde{\bold{z}}_1^2]=\frac{1}{N}\sum_{n=1}^Nz_{1,n}^2=-\mathcal{L(\bold{w}_1)}+CV[z~1​]=E[z~12​]=N1​n=1∑N​z1,n2​=−L(w1​)+C

From this we see that minimizing the reconstruction error is equivalent to maximizing the variance of the projected data.

This is why it is often said that PCA finds the direction of maximal variance.

20.1.2.3 Induction step

Now let us find another direction w2\bold{w}_2w2​ to further minimize the reconstruction error, subject to w1⊤w2=0\bold{w}_1^\top \bold{w}_2=0w1⊤​w2​=0 and w2⊤w2=1\bold{w}_2^\top \bold{w}_2=1w2⊤​w2​=1:

L(w1,z~1,w2,z~2)=1N∑n=1N∣∣xn−z1,nw1−z2,nw2∣∣22\mathcal{L}(\bold{w}_1,\tilde{\bold{z}}_1,\bold{w}_2,\tilde{\bold{z}}_2)=\frac{1}{N}\sum_{n=1}^N||\bold{x}_n-z_{1,n}\bold{w}_1-z_{2,n}\bold{w}_2||_2^2L(w1​,z~1​,w2​,z~2​)=N1​n=1∑N​∣∣xn​−z1,n​w1​−z2,n​w2​∣∣22​

Optimizing wrt w1\bold{w}_1w1​ and z~1\tilde{\bold{z}}_1z~1​ gives the same solution as before.

We can show that ∂L∂w2=0\frac{\partial \mathcal{L}}{\partial \bold{w}_2}=0∂w2​∂L​=0 yields z2,n=w2⊤xnz_{2,n}=\bold{w}_2^\top\bold{x}_nz2,n​=w2⊤​xn​.

Substituting this solution yields:

L(w2)=1N∑n=1Nxn⊤xn−w1⊤xnxn⊤w1−w2⊤xnxn⊤w2=C−w2⊤Σ^w2\begin{align} \mathcal{L}(\bold{w}_2)&=\frac{1}{N}\sum_{n=1}^N \bold{x}_n^\top \bold{x}_n-\bold{w}_1^\top \bold{x}_n \bold{x}_n^\top \bold{w}_1-\bold{w}_2^\top \bold{x}_n \bold{x}_n^\top \bold{w}_2 \\ &=C-\bold{w}_2^\top \hat{\Sigma}\bold{w}_2 \end{align}L(w2​)​=N1​n=1∑N​xn⊤​xn​−w1⊤​xn​xn⊤​w1​−w2⊤​xn​xn⊤​w2​=C−w2⊤​Σ^w2​​​

Dropping the constant term, plugging the optimal w1\bold{w}_1w1​ and adding the constraint yields:

L~(w2)=−w2⊤Σ^w2+λ1(w2⊤w2−1)+λ2(w1⊤w2−0)\tilde{\mathcal{L}}(\bold{w}_2)=-\bold{w}_2^\top\hat{\Sigma}\bold{w}_2+\lambda_1(\bold{w}_2^\top\bold{w}_2-1)+\lambda_2(\bold{w}_1^\top\bold{w}_2-0)L~(w2​)=−w2⊤​Σ^w2​+λ1​(w2⊤​w2​−1)+λ2​(w1⊤​w2​−0)

The solution is given by the eigenvector with the second largest eigenvalue:

Σ^w2=λ2w2\hat{\Sigma}\bold{w}_2=\lambda_2\bold{w}_2Σ^w2​=λ2​w2​

The proof continue that way to show that W^=UL\hat{W}=U_LW^=UL​.

20.1.3 Computational issues

20.1.3.1 Covariance matrix vs Correlation matrix

We have been working with the covariance matrix, but it’s better to work with the correlation matrix because PCA might be “mislead” by the directions in which the variance is higher because of the measurement scale.

20.1.3.2 Dealing with high-dimensional data

We have introduced PCA as finding the eigenvectors of the covariance matrix X⊤X∈RD×DX^\top X \in \R^{D\times D}X⊤X∈RD×D.

When N<DN<DN<D, it is faster to work with the Gram matrix XX⊤∈RN×NXX^\top \in \R^{N\times N}XX⊤∈RN×N instead. We show how to use it.

Let UUU be the eigenvectors of XX⊤XX^\topXX⊤ with corresponding eigenvalues Λ\LambdaΛ. By definition, we have:

(XX⊤)U=UΛ(XX^\top) U = U\Lambda (XX⊤)U=UΛ

so by left-multiplying by X⊤X^\topX⊤we get:

(X⊤X)(X⊤U)=(X⊤U)Λ(X^\top X)(X^\top U)=(X^\top U)\Lambda(X⊤X)(X⊤U)=(X⊤U)Λ

from which we see that the eigenvectors of X⊤XX^\top XX⊤X are V=X⊤UV=X^\top UV=X⊤U, with the same eigenvalues Λ\LambdaΛ.

However, these eigenvectors are not normalized, since:

∣∣vj∣∣2=uj⊤XX⊤uj=λjuj⊤uj=λj||\bold{v}_j||^2=\bold{u}_j^\top X X^\top \bold{u}_j=\lambda_j \bold{u}^\top_j\bold{u}_j=\lambda_j∣∣vj​∣∣2=uj⊤​XX⊤uj​=λj​uj⊤​uj​=λj​

The normalized eigenvector are given by:

V=X⊤UΛ−1/2V=X^\top U \Lambda^{-1/2}V=X⊤UΛ−1/2

This provides an alternative way to compute the PCA basis, it also allows us to use to kernel trick.

20.1.3.3 Computing PCA using SVD

We now show the equivalence between the PCA computed using eigenvectors and the truncated SVD.

Let UΣΛΣUΣ⊤U_\Sigma \Lambda_\Sigma U_\Sigma^\topUΣ​ΛΣ​UΣ⊤​ be the top LLL engendecomposition of the covariance matrix Σ∝X⊤X\Sigma \propto X^\top XΣ∝X⊤X (assuming XXX is centered).

We know that the optimal estimate of the projection is W=UΣW=U_\SigmaW=UΣ​.

Now let UXSXVX⊤U_X S_XV^\top _XUX​SX​VX⊤​ be the LLL-truncated SVD approximation of the matrix XXX.

We know that the right singular vectors of XXX are the eigenvectors of X⊤XX^\top XX⊤X, so VX=UΣ=WV_X =U_\Sigma=WVX​=UΣ​=W.

In addition, the singular values of XXX are related to the eigenvalues of X⊤XX^\top XX⊤X with:

λk=sk2/N\lambda_k=s_k^2/Nλk​=sk2​/N

Now suppose we are interested in the principal components, we have:

Z=XW=UXSXVX⊤VX=UXSXZ=XW=U_XS_XV_X^\top V_X=U_XS_XZ=XW=UX​SX​VX⊤​VX​=UX​SX​

Finally, if we want to reconstruct the data:

X^=ZW⊤=UXSXVX⊤\hat{X}=ZW^\top =U_XS_XV_X^\topX^=ZW⊤=UX​SX​VX⊤​

This is precisely the same as truncated SVD approximation.

Thus, we see that we can perform PCA either using a eigendecomposition of X⊤XX^\top XX⊤X or a SVD decomposition of XXX. The latter is often preferable for computational reason.

For very high dimensional problem, we can use a randomized SVD algorithm.

20.1.4 Choosing the number of latent dimensions

20.1.4.1 Reconstruction error

Let us define the reconstruction error on some dataset D\mathcal{D}D when using LLL dimensions:

LL=1∣D∣∑n∈D∣∣xn−x^n∣∣22\mathcal{L}_L=\frac{1}{|\mathcal{D}|}\sum_{n\in \mathcal{D}} ||\bold{x}_n-\bold{\hat{x}}_n||^2_2LL​=∣D∣1​n∈D∑​∣∣xn​−x^n​∣∣22​

where the reconstruction is given by x^n=Wzn+μ\bold{\hat{x}}_n=W\bold{z}_n+\mux^n​=Wzn​+μ and zn=W⊤(x^n−μ)\bold{z}_n=W^\top (\bold{\hat{x}}_n-\mu)zn​=W⊤(x^n​−μ).

We see that it drops quite quickly after, indicating that we capture most of the empirical correlation with a small number of factors.

Of course, if we choose L=rank(X)L=\mathrm{rank}(X)L=rank(X), we get zero reconstruction error on the training set. To avoid overfitting, we compute the reconstruction error on the test set.

Here we see that the error continues to decrease even when the model gets more complex. Thus, we don’t see the typical U-shaped curve that we see in supervised learning.

The problem is that PCA is not a proper generative model of the data: if you give it more latent dimensions, it will approximate the test data more accurately. A similar problem arises if we plot reconstruction error on the test set with K-means clustering.

20.1.4.2 Scree plot

A common alternative to plotting reconstruction error vs LLL is to plot the eigenvalues in decreasing order of magnitude.

We can show that:

LL=∑j=L+1Dλj\mathcal{L}_L=\sum_{j=L+1}^{D}\lambda_jLL​=j=L+1∑D​λj​

Thus as the number of dimensions increases, the eigenvalues get smaller and so do the reconstruction error.

An related quantity is the fraction of variance explained:

FL=∑j=1Lλj∑j=1LmaxλjF_L=\frac{\sum_{j=1}^L \lambda_j}{\sum_{j=1}^{L_\mathrm{max}} \lambda_j}FL​=∑j=1Lmax​​λj​∑j=1L​λj​​

20.1.3.4 Profile likelihood

Although there is no U-shape in the reconstruction error plot, there is sometimes an “elbow” where the error suddenly changes from relatively large to relatively small.

The idea is that, for L<L∗L<L^*L<L∗ where L∗L^*L∗ is the “true” latent dimensionality, the rate of decrease of errors will be high, whereas for L>L∗L>L^*L>L∗ the gains will be smaller since the model is already complex enough to capture the true distribution.

One way to automate the detection of the change of gradient in the curve is to profile likelihood.
