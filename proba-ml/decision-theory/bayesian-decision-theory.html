5.1 Bayesian Decision Theory

5.1 Bayesian decision theory

Bayesian inference allows us to update our belief about hidden quantities HHH based on observed data x∈Xx\in Xx∈X. To convert this inference into an action, we use Bayesian decision theory.

5.1.1 Basics

We assume an agent has a set of possible actions A\mathcal{A}A. Each of these actions have a cost and benefits, depending on some state H∈HH \in\mathcal{H}H∈H.

We can encode this information using a loss function ℓ(h,a)\ell(h, a)ℓ(h,a), specifying the cost of taking action a∈Aa\in\mathcal{A}a∈A with state h∈Hh\in\mathcal{H}h∈H.

In medical circles, a common measure to quantify the benefits of a drug is the quality-adjusted life years (QALY)

Once we have the loss function, we can compute the posterior expected loss or risk

R(a∣x)≜Ep(h∣x)[ℓ(h,a)]=∑h∈Hℓ(h,a)p(h∣x)R(a|x) \triangleq \mathbb{E}_{p(h|x)}[\ell(h,a)]=\sum_{h\in \mathcal{H}} \ell(h,a)p(h|x)R(a∣x)≜Ep(h∣x)​[ℓ(h,a)]=h∈H∑​ℓ(h,a)p(h∣x)

The optimal policy specifies what action to take given observed data xxx

π∗(x)=arg min⁡a∈AR(a∣x)\pi^*(x)= \argmin_{a\in\mathcal{A}} R(a|x)π∗(x)=a∈Aargmin​R(a∣x)

In the case of Covid, let say that xxx is the age of each patient. We can convert test results into a distribution over disease states, and by combining it with the loss matrix above we can compute the optimal policy for each observation.

Let say we improve the precision of the test, the probability that a positive test is correct increases and the cost of no-op for elderly increases as well, thus changing the action to take into 1.

5.1.2 Classification problems

Given an observed input xxx, what is the optimal class label to predict?

5.1.4 PR Curve

In rank retrieval problems, a PR curve can be non-monotonic (opens in a new tab). Suppose a classifier has a precision of 90% with recall of 10%, and a precision of 96% with a recall of 20%. We can compute the maximum precision (96%) we can achieve with at least a recall of KKK (10%): this is the interpolated precision.

The average of the interpolated precisions (AP) is the area under the interpolated PR curve.

The mean average precision (mAP) is the mean of the AP over a set of different PR curves.

The FβF_\betaFβ​ score can be defined as follow for β>0\beta>0β>0:

1Fβ=β2β2+11R+11+β21P\frac{1}{F_\beta}=\frac{\beta^2}{\beta^2+1}\frac{1}{\mathcal{R}}+\frac{1}{1+\beta^2}\frac{1}{\mathcal{P}}Fβ​1​=β2+1β2​R1​+1+β21​P1​

F1=2P.RP+RF_1=\mathcal{\frac{2P.R}{P+R}}F1​=P+R2P.R​

The harmonic mean is more conservative than the arithmetic mean and it requires both precision and recall to be high.

Suppose the recall is very high and the precision is very low, we would have P+R2≈R/2≈50%\frac{\mathcal{P+R}}{2} \approx R/2 \approx50\%2P+R​≈R/2≈50%

whereas F1≈2P.R/R≈2P<<50%F_1 \approx\mathcal{2P.R/R\approx2P<< 50\%}F1​≈2P.R/R≈2P<<50%

In regression settings, the set of state and action space is H=A=R\mathcal{H=A}=\mathbb{R}H=A=R

L2 loss

The most common loss is the ℓ2\ell_2ℓ2​ loss:

ℓ2(h,a)=(h−a)2\ell_2(h, a)=(h-a)^2ℓ2​(h,a)=(h−a)2

In this case, the risk is given by:

R(a∣x)=E[(h−a)2∣x]=E[h2∣x]−2aE[h∣x]+a2R(a|x)=\mathbb{E}[(h-a)^2|x]=\mathbb{E}[h^2|x]-2a\mathbb{E}[h|x]+a^2R(a∣x)=E[(h−a)2∣x]=E[h2∣x]−2aE[h∣x]+a2

The optimal action reduce the risk at that point to 0, in this case the posterior mean:

∂∂aR(a∣x)=−2E[h∣x]+2a=0⇒π(x)=E[h∣x]=∫hp(h∣x)dh\frac{\partial}{\partial a}R(a|x)=-2\mathbb{E}[h|x]+2a=0 \Rightarrow \pi(x)=\mathbb{E}[h|x]=\int hp(h|x)dh∂a∂​R(a∣x)=−2E[h∣x]+2a=0⇒π(x)=E[h∣x]=∫hp(h∣x)dh

5.1.6 Probabilistic prediction problems

We now assume the set of possible actions is to pick a probability distribution over some value. The state is a distribution h=p(Y∣x)h=p(Y|x)h=p(Y∣x) and the action is another a=q(Y∣x)a=q(Y|x)a=q(Y∣x).

We want to pick qqq to minimize E[ℓ(p,q)]\mathbb{E}[\ell(p,q)]E[ℓ(p,q)] for a given xxx.

Kullback Leibler divergence

DKL(p∣∣q)≜∑y∈Yp(y)log⁡p(y)q(y)=∑p(y)log⁡p(y)−∑p(y)log⁡q(y)=−H(p)+H(p,q)\begin{align} D_{KL}(p||q)&\triangleq\sum_{y\in\mathcal{Y}} p(y)\log\frac{p(y)}{q(y)} \\ &= \sum p(y)\log p(y) -\sum p(y) \log q(y) \\ &= -\mathbb{H}(p) + \mathbb{H}(p,q) \end{align}DKL​(p∣∣q)​≜y∈Y∑​p(y)logq(y)p(y)​=∑p(y)logp(y)−∑p(y)logq(y)=−H(p)+H(p,q)​​

H(p)\mathbb{H}(p)H(p) and H(p,q)\mathbb{H}(p,q)H(p,q) are the entropy and cross-entropy.

yyy is discrete, but this can be generalized to continuous values.

To find the optimal distribution to use, we minimize DKL(p∣∣q)D_{KL}(p||q)DKL​(p∣∣q) and as H(p)\mathbb{H}(p)H(p) is constant wrt qqq, we only minimize the cross-entropy term:

q∗(Y∣x)=arg min⁡qH(p(Y∣x),q(Y∣x))q^*(Y|x)= \argmin_q \mathbb{H}(p(Y|x),q(Y|x))q∗(Y∣x)=qargmin​H(p(Y∣x),q(Y∣x))

If the state is degenerate and puts all its mass on a single outcome i.e. h=p(Y∣x)=I(y=c)h=p(Y|x)=\mathbb{I}(y=c)h=p(Y∣x)=I(y=c), the cross entropy becomes:

H(δ(y=c),q)=−∑yδ(y=c)log⁡q(y)=−log⁡q(c)\mathbb{H}(\delta(y=c),q)=-\sum_y\delta(y=c)\log q(y)=-\log q(c)H(δ(y=c),q)=−y∑​δ(y=c)logq(y)=−logq(c)

This is the log loss of the predictive distribution qqq when given target label ccc.

Proper scoring rule

The key property we desire is that the loss function is minimized iff the chosen distribution qqq matches the true distribution ppp, ie ℓ(p,p)≤ℓ(p,q)\ell(p,p)\leq \ell(p,q)ℓ(p,p)≤ℓ(p,q) $$, ℓ\ellℓ is the proper scoring rule.

The cross-entropy is a proper scoring rule since DKL(p∣∣p)≤DKL(p∣∣q)D_{KL}(p||p)\leq D_{KL}(p||q)DKL​(p∣∣p)≤DKL​(p∣∣q), but the log⁡p(y)/q(y)\log p(y)/q(y)logp(y)/q(y) term is sensitive to errors for low probability events

A common alternative is the Brier score:

ℓ(p,q)≜1C∑c=1C(q(y=c∣x)−p(y=c∣x))2\ell(p,q) \triangleq\frac{1}{C}\sum_{c=1}^C\big(q(y=c|x)-p(y=c|x)\big)^2ℓ(p,q)≜C1​c=1∑C​(q(y=c∣x)−p(y=c∣x))2

See this discussion on scikit-learn (opens in a new tab) about the benefit and drawbacks of Log loss vs Brier score.
