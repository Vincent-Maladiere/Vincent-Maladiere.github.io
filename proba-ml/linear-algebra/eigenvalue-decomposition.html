7.4.1 Basics

For a square matrix A∈Rn×nA\in \mathbb{R}^{n\times n}A∈Rn×n, λ∈R\lambda\in\mathbb{R}λ∈R is an eigenvalue of AAA if:

Au=λu,    u≠0A\bold{u}=\lambda \bold{u},\;\;\bold{u}\neq\bold{0}Au=λu,u=0

with u∈Rn\bold{u}\in \mathbb{R}^nu∈Rn eigenvector.

Intuitively, multiplying u\bf uu by AAA results in a vector in the same direction that u\bf uu, scaled by λ.\lambda.λ.

Since cuc\bold{u}cu is also an eigenvector of λ\lambdaλ, we consider eigenvectors to be normalized with length 1.

The equation above gives us:

Au−λu=(A−λI)u=0A\bold{u}-\lambda \bold{u}=(A-\lambda I)\bold{u}=0Au−λu=(A−λI)u=0

Therefore, this has a non-zero solution iff A−λIA-\lambda IA−λI has a non-empty nullspace, which is only the case if this matrix is singular:

det(A−λI)=0\mathrm{det}(A-\lambda I)=0det(A−λI)=0

Properties:

tr(A)=∑i=1nλidet(A)=∏i=1nλi\begin{align} \mathrm{tr}(A)&=\sum_{i=1}^n\lambda_i \\ \mathrm{det}(A)&=\prod_{i=1}^n\lambda_i \end{align}tr(A)det(A)​=i=1∑n​λi​=i=1∏n​λi​​​
  • The rank of AAA is equal to its number of non-zero eigenvalues
  • If AAA is non-singular, 1/λi1/\lambda_i1/λi​ is an eigenvalue of A−1A^{-1}A−1 with corresponding eigenvectors: A−1ui=1/λiuiA^{-1}\bold{u}_i=1/\lambda_i \bold{u}_iA−1ui​=1/λi​ui​
  • The eigenvalues of a triangular matrix are just its diagonal entries.

7.4.2 Diagonalization

We can write all eigenvectors equations simultaneously:

AU=UΛAU=U\LambdaAU=UΛ

with U∈Rn×,nU\in\mathbb{R}^{n\times, n}U∈Rn×,n and Λ=diag(λ1,…,λn)\Lambda=\mathrm{diag}(\lambda_1, \dots,\lambda_n)Λ=diag(λ1​,…,λn​)

If the eigenvectors are linearly independent, then UUU is invertible and:

A=UΛU−1A=U\Lambda U^{-1}A=UΛU−1

A matrix that can be written in this form is called diagonalizable.

7.4.3 Eigenvalues of symmetric matrices

When AAA is real and symmetric, it can be shown that its eigenvalues are real and its eigenvectors are orthonormal:

ui⊤uj=0,  i≠jui⊤ui=1\begin{align} u^\top_i u_j&=0, \;i\neq j \\ u^\top_i u_i&=1 \\ \end{align}ui⊤​uj​ui⊤​ui​​=0,i=j=1​​

So in matrix notation: U⊤U=UU⊤=IU^\top U=UU^\top=IU⊤U=UU⊤=I, hence UUU is an orthogonal matrix.

We can represent AAA as:

A=UΛU⊤=∑i=1nλiuiui⊤A=U\Lambda U^\top=\sum_{i=1}^n\lambda_i \bold{u}_i\bold{u}_i^\topA=UΛU⊤=i=1∑n​λi​ui​ui⊤​

Thus multiplying by any symmetrical matrix AAA can be viewed as multiplying by a rotation matrix UUU, a scaling matrix Λ\LambdaΛ and inverse rotation with U⊤U^\topU⊤.

Since U⊤=U−1U^\top=U^{-1}U⊤=U−1, AAA is easily invertible:

A−1=UΛ−1U⊤=∑i=1nλi−1uiui⊤A^{-1}=U\Lambda^{-1}U^\top=\sum_{i=1}^n \lambda_i^{-1} \bold{u}_i \bold{u}_i^\topA−1=UΛ−1U⊤=i=1∑n​λi−1​ui​ui⊤​

7.4.5 Standardizing and whitening data

We can standardize X∈RN×DX\in\mathbb{R}^{N\times D}X∈RN×D so that each column has a mean of 0 and a variance of 1. To remove the correlations between the columns, we need to whiten the data.

Let Σ=1nX⊤X=EDE⊤\Sigma=\frac{1}{n}X^\top X=EDE^\topΣ=n1​X⊤X=EDE⊤ be the empirical covariance matrix and its diagonalization.

We can define the PCA whitening matrix as:

Wpca=D−1/2E⊤W_{pca}=D^{-1/2}E^\topWpca​=D−1/2E⊤

Let y=Wpcax\bold{y}=W_{pca} \bold{x}y=Wpca​x, we can check that:

Cov[y]=WE[xx⊤]W⊤=WΣW⊤=D—1/2E⊤(EDE⊤)ED−1/2=I\mathrm{Cov}[\bold{y}]=WE[\bold{xx}^\top]W^\top=W \Sigma W^\top=D^{—1/2}E^\top (EDE^\top)E D^{-1/2}=ICov[y]=WE[xx⊤]W⊤=WΣW⊤=D—1/2E⊤(EDE⊤)ED−1/2=I

The whitening matrix is not unique, since any rotation W=RWpcaW=RW_{pca}W=RWpca​ will maintain the whitening property Σ−1=W⊤W\Sigma^{-1} =W^\top WΣ−1=W⊤W.

For example, with R=E:R=E:R=E:

Wzca=ED−1/2E⊤=Σ−1/2=VS—1V⊤W_{zca}=ED^{-1/2}E^\top=\Sigma^{-1/2}=VS^{—1}V^\topWzca​=ED−1/2E⊤=Σ−1/2=VS—1V⊤

with, U,S,VU,S,VU,S,V the SVD decomposition of XXX.

This is called the Mahalanobis whitening or ZCA (zero-phase component analysis). The advantage of ZCA over PCA is that the resulting matrix is closer to the original matrix —in the least square sense.

When applied to images, the ZCA vectors still look like images.

7.4.6 Power method

This method is useful to compute the distribution of massive transition matrices (like Google PageRank).

Let AAA be a matrix with orthonormal eigenvectors ui\bold{u}_iui​ and eigenvalues ∣λ1∣>⋯≥∣λm∣≥0|\lambda_1|>\dots\geq|\lambda_m|\geq0∣λ1​∣>⋯≥∣λm​∣≥0

Let v(o)v_{(o)}v(o)​ be an arbitrary vector in the range of AAA, such that Ax=v(o)A\bold{x}=v_{(o)}Ax=v(o)​.

We have:

v0=U(ΛU⊤x)=a1u1+⋯+amumv_{0}=U(\Lambda U^\top x)=a_1 \bold{u}_1 +\dots+a_m\bold{u}_mv0​=U(ΛU⊤x)=a1​u1​+⋯+am​um​

So by multiplying with AAA and renormalizing at each iteration:

vt∝Avt−1v_{t}\propto Av_{t-1}vt​∝Avt−1​

So:

vt∝a1λ1tu1+⋯+amλmtum=λ1t(a1u1+⋯+am(λmλ1)tum)→λ1ta1u1\begin{align} v_{t}&\propto a_1 \lambda_1^t\bold{u}_1+\dots+a_m\lambda_m^t\bold{u}_m \\ &= \lambda_1^t\big(a_1\bold{u}_1+\dots+a_m(\frac{\lambda_m}{\lambda_1})^t\bold{u}_m\big) \\ &\rightarrow \lambda_1^ta_1 \bold{u}_1 \end{align}vt​​∝a1​λ1t​u1​+⋯+am​λmt​um​=λ1t​(a1​u1​+⋯+am​(λ1​λm​​)tum​)→λ1t​a1​u1​​​

To compute λ1\lambda_1λ1​, we define the Reyleight coefficient:

R(A,ui)≜ui⊤Auiui⊤ui=λiui⊤uiui⊤ui=λiR(A,\bold{u}_i)\triangleq\frac{\bold{u}_i^\top A \bold{u}_i}{\bold{u}_i^\top \bold{u}_i}=\frac{\lambda_i\bold{u}^\top_i\bold{u}_i}{\bold{u}_i^\top\bold{u}_i}=\lambda_iR(A,ui​)≜ui⊤​ui​ui⊤​Aui​​=ui⊤​ui​λi​ui⊤​ui​​=λi​

Therefore in our case:

R(A,vt)=λ1R(A,v_{t})=\lambda_1R(A,vt​)=λ1​

7.4.7 Deflation

Suppose we have computed λ1\lambda_1λ1​ using the power method, we now compute the subsequent λi\lambda_iλi​.

We project out the u1\bold{u}_1u1​ components from AAA:

A(2)=(I−u1u1⊤)A(1)=A(1)−λ1u1u1⊤A^{(2)}=(I-\bold{u}_1\bold{u}_1^\top)A^{(1)}=A^{(1)}-\lambda_1 \bold{u}_1\bold{u}_1^\topA(2)=(I−u1​u1⊤​)A(1)=A(1)−λ1​u1​u1⊤​

This is called matrix deflation. We can now run the power method on A(2)A^{(2)}A(2) to find λ2\lambda_2λ2​, the largest eigenvalue on the subspace orthogonal to u1\bold{u}_1u1​.

Deflation can be used to implement PCA, by computing the first KKK eigenvectors of a covariance matrix.

7.4.8 Eigenvectors optimize quadratic forms

Consider the following constrained optimization problem:

max⁡x∈Rn  x⊤Ax,      subject  to  ∣∣x∣∣22=1\max_{x\in\mathbb{R}^n}\;\bold{x}^\top A\bold{x}, \;\;\;\mathrm{subject\;to\;}||\bold{x}||^2_2=1x∈Rnmax​x⊤Ax,subjectto∣∣x∣∣22​=1

for a symmetric matrix A∈SnA\in\mathbb{S}^nA∈Sn.

A standard way to solve a constrained optimization problem is by forming the Lagrangian:

L(x,λ)=x⊤Ax+λ(1−x⊤x)\mathcal{L}(\bold{x},\lambda)=\bold{x}^\top A\bold{x}+\lambda(1-\bold{x}^\top \bold{x})L(x,λ)=x⊤Ax+λ(1−x⊤x)

Therefore:

∇xL(x,λ)=2A⊤x−2λx=0\nabla_x\mathcal{L}(\bold{x},\lambda)=2A^\top\bold{x}-2\lambda \bold{x}=0∇x​L(x,λ)=2A⊤x−2λx=0

So the problem is reduced to Ax=λxA\bold{x}=\lambda \bold{x}Ax=λx, and the only solutions to maximize x⊤Ax\bold{x}^\top A\bold{x}x⊤Ax given x⊤x=1\bold{x}^\top\bold{x}=1x⊤x=1 are the eigenvectors of AAA.
