7.1 Introduction

7.1.1 Notation

Vector

A vector x∈Rn\bold{x}\in \mathbb{R}^nx∈Rn is usually written as a column vector:

x=[x1⋮xn]\bold{x}=\begin{bmatrix}x_1 \\ \vdots \\x_n\end{bmatrix}x=​x1​⋮xn​​​

The vector of all ones is 1\bold{1}1, of all zeros is 0\bold{0}0.

The unit or one-hot vector ei\bold{e}_iei​ is a vector of all zeros except entry iii which is 111.

ei=(0,…,0,1,0,…,0)e_i=(0,\dots, 0,1,0,\dots,0)ei​=(0,…,0,1,0,…,0)

Matrix

A matrix A∈Rn×m\bold{A} \in \mathbb{R}^{n\times m}A∈Rn×m is defined similarly. If n=mn=mn=m the matrix is square.

The transpose of a matrix results in flipping its row and columns: (AT)ij=Aji(A^T)_{ij}=A_{ji}(AT)ij​=Aji​. Some of its properties:

  • (AT)T=A(A^T)^T=A(AT)T=A
  • (A+B)T=AT+BT(A+B)^T=A^T+B^T(A+B)T=AT+BT
  • (AB)T=BTAT(AB)^T=B^TA^T(AB)T=BTAT

A symmetric square matrices satisfies AT=AA^T=AAT=A. The set of all symmetric matrices of size nnn is denoted Sn\Bbb S^nSn.

Tensor

A tensor is a generalization of a 2d array:

The rank of a tensor is its number of dimensions.

We can stack rows (resp columns) of a matrix horizontally (resp vertically) to get a C-contiguous or row-major order (resp F-contiguous or columns-major order).

7.1.2 Vector spaces

Spans

We can view a vector x∈Rn\bold{x} \in \Bbb R^nx∈Rn as a point in n-dimensional Euclidean space. A vector space is a collection of such vector, which can be added together and scale by scalar c∈Rc\in\Bbb Rc∈R.

A set of vector {x1,...,xn}\{\bold{x}_1,...,\bold{x}_n\}{x1​,...,xn​} is linearly independent if no vector can be represented as a linear combination of the others.

The span of {x1,...,xn}\{\bold{x}_1,...,\bold{x}_n\}{x1​,...,xn​} is the set of all vectors that can be obtained its linear combinations:

span({x1,...,xn})≜{v:v=∑i=1nαixi  ,αi∈R}\mathrm{span}(\{\bold{x}_1,...,\bold{x}_n\})\triangleq \Big\{v:v=\sum_{i=1}^n \alpha_i \bold{x}_i \;,\alpha_i\in\Bbb R \Big\}span({x1​,...,xn​})≜{v:v=i=1∑n​αi​xi​,αi​∈R}

It can be shown that if each xi∈Rn\bold{x_i} \in \mathbb{R}^nxi​∈Rn form a set of linearly independent vectors, then:

span({x1,...,xn})=Rn\mathrm{span}(\{\bold{x}_1,...,\bold{x}_n\})=\mathbb{R}^nspan({x1​,...,xn​})=Rn

and therefore any vectors of Rn\mathbb{R}^nRn can be written as a linear combination of this set of vector.

A basis B\mathcal{B}B is a set of linearly independent vector that spans the whole space:

span(B)=Rn\mathrm{span}(\mathcal{B})=\R^nspan(B)=Rn

There are often multiple basis to choose from, a standard one is the coordinate vectors ei\bold{e}_iei​.

Linear maps

A linear map is the transformation f:V→Wf:\mathcal{V\rightarrow W}f:V→W such that:

  • f(v+w)=f(v)+f(w),  ∀v,w∈Vf(v+w)=f(v)+f(w),\; \forall v,w\in\mathcal{V}f(v+w)=f(v)+f(w),∀v,w∈V
  • f(av)=af(v)f(av)=af(v)f(av)=af(v)

Once the basis of V\mathcal{V}V is chosen, a linear map is completely determined by computing the image basis of W\mathcal{W}W.

Suppose V=Rn,W=Rm,\mathcal{V}=\mathbb{R}^n,\mathcal{W}=\mathbb{R}^m,V=Rn,W=Rm, we can compute f(xi)∈Rmf(\bold{x}_i)\in\mathbb{R}^mf(xi​)∈Rm and store it in A∈Rm×n\bold{A} \in \mathbb{R}^{m \times n}A∈Rm×n.

We can then compute any y∈Rmy\in \mathbb{R}^my∈Rm for any x∈Rn\bold{x}\in\mathbb{R}^nx∈Rn by:

y=(∑i=1na1,ixi,...,∑i=1nam,ixi)=Axy=\Big(\sum_{i=1}^n a_{1,i} x_i, ...,\sum_{i=1}^n a_{m,i} x_i \Big)=\bf Axy=(i=1∑n​a1,i​xi​,...,i=1∑n​am,i​xi​)=Ax

The range of A\bold{A}A the space reached by the matrix A\bold{A}A:

range(A)≜{v∈Rm:v=Ax,  x∈Rn}\mathrm{range}(\bold{A}) \triangleq\{\bold{v}\in \mathbb{R}^m:\bold{v}=\bold{Ax},\;\bold{x}\in\mathbb{R}^n\}range(A)≜{v∈Rm:v=Ax,x∈Rn}

The nullspace is:

nullspace(A)≜{x∈Rn:Ax=0}\mathrm{nullspace}(\bold{A})\triangleq \{\bold{x}\in \mathbb{R}^n:\bold{Ax=0}\}nullspace(A)≜{x∈Rn:Ax=0}

Linear projection

The projection of y∈Rm\bold{y} \in \mathbb{R}^my∈Rm onto the span {x1,…,xn}\{\bold{x}_1,\dots,\bold{x}_n\}{x1​,…,xn​} with xi∈Rm\bold{x}_i \in\mathbb{R}^mxi​∈Rm, is:

v=Proj(y,{x1,...,xn})=arg min⁡v∈span({x1,...,xn})∣∣v−y∣∣2\bold{v}=\mathrm{Proj}(\bold{y},\{\bold{x}_1,...,\bold{x}_n\})=\argmin_{\bold{v} \in \mathrm{span}(\{x_1,...,x_n\})}||\bold{v}-\bold{y}||_2v=Proj(y,{x1​,...,xn​})=v∈span({x1​,...,xn​})argmin​∣∣v−y∣∣2​

And for a full rank matrix Am×nA^{m\times n}Am×n, with m≥nm\geq nm≥n , we can define the projection of y∈Rm\bold{y}\in\mathbb{R}^my∈Rm on the range of A:A:A:

v=Proj(y,A)=arg min⁡v∈R(a)∣∣v−y∣∣2=A(A⊤A)−1A⊤y\bold{v}=\mathrm{Proj}(\bold{y},A)=\argmin_{\bold{v}\in \mathcal{R}(a)}||\bold{v-y}||_2=A(A^\top A)^{-1}A^\top yv=Proj(y,A)=v∈R(a)argmin​∣∣v−y∣∣2​=A(A⊤A)−1A⊤y

7.1.3 Vector and matrix norms

Vector norms

A norms is any function f:Rm→Rf:\mathbb{R}^m\rightarrow \mathbb{R}f:Rm→R that satisfies the following properties:

  • non-negativity: ∀x∈Rn,f(x)≥0\forall \bold{x} \in \mathbb{R}^n,f(\bold{x})\geq0∀x∈Rn,f(x)≥0
  • definiteness: f(x)=0  ⟺  x=0f(\bold{x})=0\iff \bold{x=0}f(x)=0⟺x=0
  • absolute value homogeneity: ∀t∈R,f(tx)=∣t∣f(x)\forall t\in\mathbb{R},f(t\bold{x})=|t|f(\bold{x})∀t∈R,f(tx)=∣t∣f(x)
  • triangle inequality: f(x+y)≤f(x)+f(y)f(\bold{x}+\bold{y})\leq f(\bold{x})+f(\bold{y})f(x+y)≤f(x)+f(y)

Common norms:

  • p-norm: ∣∣x∣∣p=(∑i=1N∣xi∣p)1/p||\bold{x}||_p=\big(\sum_{i=1}^N |x_i|^p\big)^{1/p}∣∣x∣∣p​=(∑i=1N​∣xi​∣p)1/p, for p≥1p\ge 1p≥1

  • ℓ1\ell_1ℓ1​ or Manhattan: ∣∣x∣∣1=∑i=1N∣xi∣||\bold{x}||_1=\sum^N_{i=1}|x_i|∣∣x∣∣1​=∑i=1N​∣xi​∣

  • ℓ2\ell_2ℓ2​ or Euclidean: ∣∣x∣∣2=∑i=1Nxi2||\bold{x}||_2=\sqrt{\sum_{i=1}^N x_i^2}∣∣x∣∣2​=∑i=1N​xi2​​

    also note the square euclidean: ∣∣x∣∣22=xx⊤||\bold{x}||^2_2=\bold{x}\bold{x}^\top∣∣x∣∣22​=xx⊤

  • max-norm or Chebyshev: ∣∣x∣∣∞=max⁡i∣xi∣||\bold{x}||_\infin=\max_i|x_i|∣∣x∣∣∞​=maxi​∣xi​∣

  • zero-norm: ∣∣x∣∣0=∑i=1NI(∣xi∣>0)||\bold{x}||_0=\sum_{i=1}^N\mathbb{I}(|x_i|>0)∣∣x∣∣0​=∑i=1N​I(∣xi​∣>0). This is a pseudo norm since it doesn’t respect homogeneity.

Matrix norms

i) Induced norms

Suppose we think of a matrix A∈Rm×nA\in\R^{m\times n}A∈Rm×n as defining a linear function f(x)=Axf(\bold{x})=A\bold{x}f(x)=Ax.

We define the induced norm of AAA as the maximum amount by which fff can lengthen input:

∣∣A∣∣p=max⁡x≠0∣∣Ax∣∣p∣∣x∣∣p=max⁡∣∣x∣∣=1∣∣Ax∣∣p||A||_p=\max_{\bold{x\ne0}} \frac{||A\bold{x}||_p}{||\bold{x}||_p}=\max_{||\bold{x||}=1}||A\bold{x}||_p∣∣A∣∣p​=x=0max​∣∣x∣∣p​∣∣Ax∣∣p​​=∣∣x∣∣=1max​∣∣Ax∣∣p​

In the special case of p=2p=2p=2 (the Euclidean norm), the induced norm is the spectral norm:

∣∣A∣∣2=λmax(A⊤A)=max⁡iσi||A||_2= \sqrt{\lambda_{max}(A^\top A)}=\max_i \sigma_i∣∣A∣∣2​=λmax​(A⊤A)​=imax​σi​

with σi\sigma_iσi​ the iiith singular value

ii) Element-wise norms

If we treat a m×nm\times nm×n matrix as a vector of size m.nm.nm.n, we can define the ppp-norm:

∣∣A∣∣p=∣∣vec(A)∣∣p=(∑i∑jaijp)1/p||A||_p=||\mathrm{vec}(A)||_p=\Big(\sum_i\sum_j a_{ij}^p\Big)^{1/p}∣∣A∣∣p​=∣∣vec(A)∣∣p​=(i∑​j∑​aijp​)1/p

when p=2p=2p=2, we get the Frobenius norm:

∣∣A∣∣F=∑i∑jaij2=tr(A⊤A)||A||_F=\sqrt{\sum_i \sum_j a_{ij}^2}=\sqrt{\mathrm{tr}(A^\top A)}∣∣A∣∣F​=i∑​j∑​aij2​​=tr(A⊤A)​

If AAA is expensive but AvAvAv is cheap, we can approximate the F-norm with the Hutchinson trace estimator:

∣∣A∣∣F2=tr(A⊤A)=E[v⊤A⊤Av]=E[∣∣Av∣∣22]||A||_F^2=\mathrm{tr}(A^\top A)=\mathbb{E}[v^\top A^\top A v]=\mathrm{E}[||Av||_2^2]∣∣A∣∣F2​=tr(A⊤A)=E[v⊤A⊤Av]=E[∣∣Av∣∣22​]

with v∼N(0,1)v \sim \mathcal{N}(0,1)v∼N(0,1)

iii) Schatten norms

The Schatten ppp-norms arise when applying the element-wise ppp-norm to the singular values of a matrix:

∣∣A∣∣p=(∑iσip(A))1/p||A||_p= \Big(\sum_i \sigma_i^p(A)\Big)^{1/p}∣∣A∣∣p​=(i∑​σip​(A))1/p

When p=1p=1p=1, we get the nuclear norm (or trace norm):

∣∣A∣∣∗=tr(A⊤A)=∑iσi=∑i∣σi∣=∣∣σ∣∣1||A||_*=\mathrm{tr}(\sqrt{A^\top A}) =\sum_i \sigma_i=\sum_i|\sigma_i|=||\sigma||_1∣∣A∣∣∗​=tr(A⊤A​)=i∑​σi​=i∑​∣σi​∣=∣∣σ∣∣1​

since σi≥0\sigma_i\geq0σi​≥0.

Using this as a regularizer encourages many singular values to become zero, resulting in a low-rank matrix.

When p=2p=2p=2, we get the Frobenius norm.

When p=∞p=\infinp=∞, we get the Spectral norm

7.1.4 Properties of a matrix

Trace

The trace of a square matrix A∈RnA\in \mathbb{R}^nA∈Rn is:

tr(A)≜∑i=1naii=∑i=1nλi\mathrm{tr}(A)\triangleq\sum_{i=1}^n a_{ii}=\sum_{i=1}^n \lambda_itr(A)≜i=1∑n​aii​=i=1∑n​λi​

We have the following:

  • tr(A⊤)=tr(A)\mathrm{tr}(A^\top)=\mathrm{tr}(A)tr(A⊤)=tr(A)
  • tr(A+B)=tr(A)+tr(B)\mathrm{tr}(A+B)=\mathrm{tr}(A)+\mathrm{tr}(B)tr(A+B)=tr(A)+tr(B)
  • tr(cA)=ctr(A)\mathrm{tr}(cA)=c\mathrm{tr}(A)tr(cA)=ctr(A)
  • tr(AB)=tr(BA)\mathrm{tr}(AB)=\mathrm{tr}(BA)tr(AB)=tr(BA)
  • tr(ABC)=tr(BCA)=tr(CAB)\mathrm{tr}(ABC)=\mathrm{tr}(BCA)=\mathrm{tr}(CAB)tr(ABC)=tr(BCA)=tr(CAB)

This last property allows to define the trace trick:

x⊤Ax=tr(x⊤Ax)=tr(xx⊤A)\bold{x}^\top A\bold{x}=\mathrm{tr}(\bold{x}^\top A \bold{x})=\mathrm{tr}(\bold{x}\bold{x}^\top A)x⊤Ax=tr(x⊤Ax)=tr(xx⊤A)

In some case, AAA might be expensive but AvAvAv cheap, we can use the Hutchinson trace approximator:

tr(A)=tr(AE[vv⊤])=E[tr(Avv⊤)]=E[tr(v⊤Av)]\mathrm{tr}(A)=\mathrm{tr}(A\mathbb{E}[vv^\top])=\mathbb{E}[\mathrm{tr}(Avv^\top)]=\mathbb{E}[\mathrm{tr}(v^\top A v)]tr(A)=tr(AE[vv⊤])=E[tr(Avv⊤)]=E[tr(v⊤Av)]

with E[vv⊤]=I\mathbb{E}[vv^\top]=\bold{I}E[vv⊤]=I

Determinant

The determinant is a measure of how much a square matrix change a unit volume when viewed as a linear transformation.

With A∈Rn×nA\in \mathbb{R}^{n\times n}A∈Rn×n:

  • ∣A∣=∣A⊤∣|A|=|A^\top|∣A∣=∣A⊤∣
  • ∣cA∣=cn∣A∣|cA|=c^n|A|∣cA∣=cn∣A∣
  • ∣AB∣=∣A∣∣B∣|AB|=|A||B|∣AB∣=∣A∣∣B∣
  • ∣A∣=0|A|=0∣A∣=0 iff AAA is singular
  • If AAA is not singular, ∣A∣−1=1/∣A∣|A|^{-1}=1/|A|∣A∣−1=1/∣A∣
  • ∣A∣=∏i=1nλi|A|=\prod_{i=1}^n\lambda_i∣A∣=∏i=1n​λi​, where λi\lambda_iλi​ are the eigenvalues of AAA

For a positive definite matrix, A=LL⊤A=LL^\topA=LL⊤ where LLL is a lower triangle Cholesky decomposition:

∣A∣=∣LL⊤∣=∣L∣2|A|=|LL^\top|=|L|^2∣A∣=∣LL⊤∣=∣L∣2

so

log⁡∣A∣=2log⁡∣L∣=2log⁡∏iLii=2tr(log⁡(diag(L)))\log |A|= 2 \log |L|= 2 \log \prod_i L_{ii} = 2 \mathrm{tr}(\log(\mathrm{diag}(L)))log∣A∣=2log∣L∣=2logi∏​Lii​=2tr(log(diag(L)))

Rank

The column (resp row) rank is the dimension spanned by the columns (resp rows) of a matrix, and its a basic fact that they are the same: rank(A)\mathrm{rank}(A)rank(A)

  • For A∈Rn×mA \in \mathbb{R}^{n \times m}A∈Rn×m, rank(A)≤min⁡(n,m)\mathrm{rank}(A)\leq \min(n,m)rank(A)≤min(n,m)

    If equality, AAA is said to be full rank (it is rank defficient otherwise).

  • rank(A)=rank(A⊤)=rank(AA⊤)=rank(A⊤A)\mathrm{rank}(A)=\mathrm{rank}(A^\top)=\mathrm{rank}(AA^\top)=\mathrm{rank}(A^\top A)rank(A)=rank(A⊤)=rank(AA⊤)=rank(A⊤A)

  • For B∈Rm×pB \in \mathbb{R}^{m\times p}B∈Rm×p, rank(AB)≤min⁡(rank(A),rank(B))\mathrm{rank}(AB)\leq \min(\mathrm{rank}(A),\mathrm{rank}(B))rank(AB)≤min(rank(A),rank(B))

  • For A,B∈Rn×mA,B \in \mathbb{R}^{n\times m}A,B∈Rn×m, rank(A+B)≤rank(A)+rank(B)\mathrm{rank}(A+B)\leq \mathrm{rank}(A)+ \mathrm{rank}(B)rank(A+B)≤rank(A)+rank(B)

  • A square matrix is invertible iff full rank

Condition number

All norms that follow are ℓ2\ell_2ℓ2​.

The condition number of a matrix measures of numerically stable any computation involving AAA will be:

κ(A)≜∣∣A∣∣.∣∣A−1∣∣≥1=σmax/σmin=λmax/λmin\begin{align} \kappa(A)&\triangleq||A||.||A^{-1}||\geq1 \\ &= \sigma_{max}/\sigma_{min}=\sqrt{\lambda_{max}/\lambda_{min}} \end{align}κ(A)​≜∣∣A∣∣.∣∣A−1∣∣≥1=σmax​/σmin​=λmax​/λmin​​​​

If κ\kappaκ is close to 1, the matrix is well conditioned (ill-conditioned otherwise).

A large condition means AAA is nearly singular. This a better measure to nearness of singularity than the size of the determinant.

For exemple, A=0.1I100×100A=0.1 \bold{I}_{100\times100}A=0.1I100×100​ leads to ∣A∣=10−100|A|=10^{-100}∣A∣=10−100 but κ(A)=1\kappa(A)=1κ(A)=1, indicating that even though AAA is nearly singular, it is well conditioned and AxA\bold{x}Ax simply scales the entry of x\bold{x}x by 0.1.

7.1.5 Special types of matrices

Diagonal matrices All elements are 0 except the diagonal.

diag(A)=diag(d1,...,dn)\mathrm{diag}(A)=\mathrm{diag}(d_1,...,d_n)diag(A)=diag(d1​,...,dn​)

We can convert a vector to a diagonal matrix with this notation

Identity matrix I\bold{I}I Diagonal of ones, with the identity property:

AI=IA=AA\bold{I}=\bold{I}A=AAI=IA=A

Block diagonal matrix

Non zero only on the main diagonal:

[A00B]\begin{bmatrix} \bold{A} &\bold{0} \\ \bold{0} &\bold{B} \end{bmatrix}[A0​0B​]

Band-diagonal matrix Non-zero entries along k-sides of the diagonal:

Triangular matrix Upper (resp lower) have only non-zero on the diagonal and above (resp below)

Useful property: the diagonal is the eigenvalues, therefore ∣L∣=∏iLii|L|=\prod_i L_{ii}∣L∣=∏i​Lii​

Positive definite matrices

A symmetric matrix A∈SnA\in\mathbb{S}^nA∈Sn is positive definite iff:

x⊤Ax>0\bold{x}^\top A \bold{x} >0x⊤Ax>0

If equality is possible, we call AAA positive semi-definite.

The set of all positive definite matrices is denoted S++n\mathbb{S}^n_{++}S++n​

A matrix can also be negative definite or indefinite: there exists x1\bold{x}_1x1​ and x2\bold{x}_2x2​ such that:

x1⊤Ax1>0  and  x2⊤Ax2<0\bold{x}_1^\top A\bold{x_1} >0 \;\mathrm{and}\;\bold{x_2} ^\top A \bold{x}_2 <0x1⊤​Ax1​>0andx2​⊤Ax2​<0

Note that a matrix having all positive coefficient doesn’t imply being positive definite ex: [4332]\begin{bmatrix}4 & 3 \\ 3 & 2\end{bmatrix}[43​32​]

Conversely, it can have negative coefficient.
