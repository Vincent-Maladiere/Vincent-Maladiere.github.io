7.2 Matrix multiplication

The product of A∈Rm×nA\in\mathbb{R}^{m \times n}A∈Rm×n and B∈Rn×pB\in\mathbb{R}^{n \times p}B∈Rn×p is AB=CAB=CAB=C with:

Cij=∑k=1nAikBkjC_{ij}=\sum_{k=1}^n A_{ik}B_{kj}Cij​=k=1∑n​Aik​Bkj​

Note that the number of columns of AAA must match the number of rows of BBB.

Matrix multiplication complexity averages O(mnp)O(mnp)O(mnp), faster methods exists (like BLAS) and hardware parallelizing the computation, like TPU or GPU.

  • Matrix multiplication is associative: A(BC)=(AB)CA(BC)=(AB)CA(BC)=(AB)C
  • Matrix multiplication is distributive: A(B+C)=AB+BCA(B+C)=AB+BCA(B+C)=AB+BC
  • In general, it is not commutative: AB≠BAAB\neq BAAB=BA

Cij=∑kAikBkj⇒AikBkjC_{ij}=\sum_k A_{ik}B_{kj} \Rightarrow A_{ik}B_{kj} Cij​=k∑​Aik​Bkj​⇒Aik​Bkj​

And for more complex operations:

Lnc=∑d∑k∑tSntkWkdVdc⇒SntkWkdVdcL_{nc}=\sum_d\sum_k\sum_t S_{ntk}W_{kd}V_{dc} \Rightarrow S_{ntk}W_{kd}V_{dc}Lnc​=d∑​k∑​t∑​Sntk​Wkd​Vdc​⇒Sntk​Wkd​Vdc​

Einsum can perform computations in an optimal order, minimizing time and memory allocation.
