11.4 Lasso regression

In ridge regression, we assumed a Gaussian prior on the parameters, which is often a good choice since it leads to smaller parameters and limits overfitting.

However, sometimes we want the parameters to be exactly 0, i.e. we want w^\hat{\bold{w}}w^ to be sparse so that we minimize the ℓ0\ell_0ℓ0​ norm:

∣∣w∣∣0=∑j=1DI(wj≠0)||\bold{w}||_0=\sum_{j=1}^D\mathbb{I}(w_j\neq 0)∣∣w∣∣0​=j=1∑D​I(wj​=0)

This is useful because it can perform feature selection, as f(x,w)=∑j=1Dxjwjf(\bold{x},\bold{w})=\sum_{j=1}^D \bold{x}_jw_jf(x,w)=∑j=1D​xj​wj​, if wj=0w_j=0wj​=0, the feature jjj is ignored.

The same idea can be applied to non-linear models (like DNN) and encourage the first layer weight to be sparse.

11.4.1 MAP estimation using a Laplace prior (ℓ1\ell_1ℓ1​ regularization)

We now focus on MAP estimation using the Laplace prior:

p(w∣λ)=∏j=1DLap(wj∣0,1/λ)∝∏j=1De−λ∣wd∣p(\bold{w}|\lambda)=\prod_{j=1}^D \mathrm{Lap}(w_j|0,1/\lambda)\propto\prod^D_{j=1}e^{-\lambda|w_d|}p(w∣λ)=j=1∏D​Lap(wj​∣0,1/λ)∝j=1∏D​e−λ∣wd​∣

where:

Lap(w∣μ,b)≜12bexp⁡(−∣w−μ∣b)\mathrm{Lap}(w|\mu,b)\triangleq \frac{1}{2b}\exp \Big({-\frac{|w-\mu|}{b}}\Big)Lap(w∣μ,b)≜2b1​exp(−b∣w−μ∣​)

Laplace put more weight on 0 than Gaussian, for the same scale.

To perform MAP estimation of a linear regression model with this prior, we minimize the objective:

PNNL(w)=−log⁡p(D∣w)−log⁡p(w∣λ)=∣∣y−Xw∣∣22+λ∣∣w∣∣1\mathrm{PNNL}(\bold{w})=-\log p(\mathcal{D}|\bold{w})-\log p(\bold{w}|\lambda)=||\bold{y}-X\bold{w}||_2^2+\lambda ||\bold{w}||_1PNNL(w)=−logp(D∣w)−logp(w∣λ)=∣∣y−Xw∣∣22​+λ∣∣w∣∣1​

This method is called lasso “least absolute shrinkage and selection operator” or ℓ1\ell_1ℓ1​ regularization.

Note we could also use the q-norm:

∣∣w∣∣q=(∑j=1D∣wj∣q)1/q||\bold{w}||_q=\Big(\sum_{j=1}^D |w_j|^q\Big)^{1/q}∣∣w∣∣q​=(j=1∑D​∣wj​∣q)1/q

For q<1q<1q<1, we can get even sparser solutions but the problem becomes non-convex. Thus, the ℓ1\ell_1ℓ1​ norm is the tightest convex relaxation of the ℓ0\ell_0ℓ0​ norm.

11.4.2 Why does the ℓ1\ell_1ℓ1​ yields sparse solutions?

The lasso objective is a non-smooth objective that can be expressed with the Lagrangian:

min⁡wNLL(w)s.t.∣∣w∣∣1≤B\min_{\bold{}w} \mathrm{NLL}(\bold{w})\quad s.t.\quad ||\bold{w}||_1\leq Bwmin​NLL(w)s.t.∣∣w∣∣1​≤B

where BBB is an upper bound to the weight. A tight bound corresponds to a high λ\lambdaλ.

From the optimization theory, we know that the optimal solution lies at the intersection of the constraint surface and the lowest level of the objective function.

When we relax the constraint BBB, we grow the constraint surface until reaching the objective. The ℓ1\ell_1ℓ1​ dice is more likely to hit the objective function on the corner rather than on the side, especially in high dimensions. The corners correspond to the sparse solution, lying on the coordinate axes.

Meanwhile, there are no corners on the ℓ2\ell_2ℓ2​ ball, it can intersect the objective function everywhere; there is no preference for sparsity.

11.4.3 Hard vs soft thresholding

One can show that the gradient for the smooth part is:

∂∂wjNLL(w)=adwd−cd\frac{\partial}{\partial w_j}\mathrm{NLL}(\bold{w})=a_dw_d-c_d∂wj​∂​NLL(w)=ad​wd​−cd​

with:

ad=∑n=1Nxnd2cd=∑n=1Nxnd(yn−w−d⊤xn,−d)\begin{align} a_d&=\sum_{n=1}^N x_{nd}^2\\ c_d&= \sum_{n=1}^N x_{nd}(y_n-\bold{w}_{-d}^\top \bold{x}_{n,-d}) \end{align}ad​cd​​=n=1∑N​xnd2​=n=1∑N​xnd​(yn​−w−d⊤​xn,−d​)​​

where w−d\bold{w}_{-d}w−d​ is w\bold{w}w without the component ddd.

Setting the gradient to zero gives the optimal update for feature ddd while keeping the other constant:

wd=cd/ad=xd⊤r−d∣∣xd∣∣22w_d=c_d/a_d=\frac{\bold{x}_d^\top \bold{r}_{-d}}{||\bold{x}_d||^2_2}wd​=cd​/ad​=∣∣xd​∣∣22​xd⊤​r−d​​

with r−d=y−X−dw−d\bold{r}_{-d}=\bold{y}-X_{-d}\bold{w}_{-d}r−d​=y−X−d​w−d​ the residuals error obtained by using all features but ddd.

Let’s add the penalty term. Unfortunately, ∣∣w1∣∣1||\bold{w}_1||_1∣∣w1​∣∣1​ is not differentiable whenever wd=0w_d=0wd​=0. We can still compute the subgradient at this point.

Depending on the value of cdc_dcd​, the solution to ∂wdL(w)=0\partial_{w_d}\mathcal{L}(\bold{w})=0∂wd​​L(w)=0 can occur at 3 different values of wdw_dwd​ as follows:

  1. If cd<−λc_d<-\lambdacd​<−λ, the feature is strongly negatively correlated with the residual, then the subgradient is zero at wd=cd+λad<0w_d=\frac{c_d+\lambda}{a_d}<0wd​=ad​cd​+λ​<0
  2. If cd∈[−λ,λ]c_d\in[-\lambda,\lambda]cd​∈[−λ,λ], the feature is weakly correlated with the residual, then the subgradient is zero at wd=0w_d=0wd​=0

In summary:

w^d=SoftThreshold(cdad,λ/ad)\hat{w}_d=\mathrm{SoftThreshold}(\frac{c_d}{a_d},\lambda/a_d)w^d​=SoftThreshold(ad​cd​​,λ/ad​)

where:

SoftThreshold(x,δ)=sign(x)max⁡(∣x∣−δ,0)\mathrm{SoftThreshold}(x,\delta)=\mathrm{sign}(x)\max(|x|-\delta,0)SoftThreshold(x,δ)=sign(x)max(∣x∣−δ,0)

The slope of the soft-thresholding shrunk line doesn’t match the diagonal (the OLS), meaning that event big coefficients are shrunk to zero.

Consequently, Lasso is a biased estimator. A simple solution known as debiasing is to first estimate which elements are non-zero using lasso, then re-estimate the chosen coefficient using least squares.

11.4.6 Variable selection consistency

ℓ1\ell_1ℓ1​ regularization is often used to perform feature selection. A method that can recover the true set of meaningful features when N→∞N\rightarrow \infinN→∞ is called model selection consistent.

Let’s create:

  • a sparse signal w∗∈RD\bold{w}^*\in\mathbb{R}^{D}w∗∈RD, consisting of 160 randomly placed ±1\pm 1±1 spikes
  • a random design matrix X∈RN×DX\in \mathbb{R}^{N\times D}X∈RN×D
  • a noisy observation y=Xw∗+ϵ\bold{y}=X\bold{w}^*+\epsilony=Xw∗+ϵ , with ϵ∼N(0,0.012)\epsilon \sim \mathcal{N}(0, 0.01^2)ϵ∼N(0,0.012)

with N=1024≤D=4096N=1024\leq D=4096N=1024≤D=4096

The second row is the ℓ1\ell_1ℓ1​ estimate using λ=0.1λmax\lambda=0.1\lambda_{max}λ=0.1λmax​. We see that it finds all coefficients w^L1\hat{\bold{w}}_{L1}w^L1​, but they are too small due to shrinkage.

The third row uses the debiasing technique and recovers the right coefficient values.

By contrast, the last row represents the signal recovered by the OLS, which is dense.

To pick the optimal value of λ\lambdaλ and perform the feature selection, we use cross-validation. It is important to note that CV will lead to good prediction accuracy, which is not necessarily the same as recovering the true model.

This is because Lasso performs selection and shrinkage i.e. the chosen coefficients are brought closer to 000, so the CV will pick a value of λ\lambdaλ that is not too large. This will result in a less sparse model with irrelevant variables.

However, various extensions to the basic method have been designed that are model selection consistent. See Statistical Learning with Sparsity (2015) (opens in a new tab)

11.4.7 Group lasso

In standard ℓ1\ell_1ℓ1​ regularization, we assume a 1:1 correspondence between a parameter and its variable. But in more complex settings, there may be many parameters associated with a single variable, so that each variable has its weight vector wd\bold{w}_dwd​ and the overall weight vector has a block structure [w1,…,wD][\bold{w}_1,\dots,\bold{w}_D][w1​,…,wD​].

To exclude the variable ddd, we have to force the entire ddd block to be 000: this is group sparsity.

11.4.7.1 Applications

  • Linear regression with categorical inputs, where we use a one-hot-encoding vector of length KKK
  • Multinomial logistic regression: The ddd’th variable will be associated with CCC different weight (one per class)
  • Multi-task learning, similarly each feature is associated with CCC different weighs (one per output task)
  • Neural networks: The kkk’th neuron will have multiple inputs, so we set all incoming weights to zero to turn it off. Group sparsity can help us learn the neural net structure.

11.4.8.1 Penalizing the two-norm

To encourage group sparsity, we partition the parameter vector into GGG groups, then we minimize:

PNNL(w)=NLL(w)+λ∑g=1G∣∣wg∣∣2\mathrm{PNNL}(\bold{w})=\mathrm{NLL}(\bold{w})+\lambda \sum_{g=1}^G ||\bold{w}_g||_2PNNL(w)=NLL(w)+λg=1∑G​∣∣wg​∣∣2​

where ∣∣w∣∣=∑d∈gwd2||\bold{w}||=\sqrt{\sum_{d\in g}w_d^2}∣∣w∣∣=∑d∈g​wd2​​ is the two-norm of the group weight vector. If the NLL is least-squares, this method is the group lasso. Note that if we had the squared of the two norms, the model would be equivalent to ridge regression.

By using the square root, we are penalizing the radius of a ball containing the group weights: the only way for the radius to be small is if all elements are small too.

Another way to see why the two-norm enforces group sparsity is to consider the gradient of the objective. If we only have one group of two parameters:

∂∂w1w12+w22=w1w12+w22\frac{\partial}{\partial w_1}\sqrt{w_1^2+w_2^2}=\frac{w_1}{\sqrt{w_1^2+w_2^2}}∂w1​∂​w12​+w22​​=w12​+w22​​w1​​
  • If w2w_2w2​ is small, the gradient will be close to 1 and the w1w_1w1​ will be shrunk to zero
  • If w1w_1w1​ is large, then if w1w_1w1​ is small the gradient will be close to 000 and w1w_1w1​ will not get updated much.

11.4.8.2 Penalizing the infinity norm

A variant of this technique replaces the two-norm with the infinity norm:

∣∣wg∣∣∞=max⁡d∈g∣wd∣||\bold{w}_g||_\infin=\max_{d\in g} |\bold{w}_d|∣∣wg​∣∣∞​=d∈gmax​∣wd​∣

This will also result in group sparsity, since if the largest element of the group is forced to zero, all the others will be zero.

We randomly pick 8 groups and assigned them to non-zero values.

11.4.8 Elastic net (ridge and lasso combined)

In group lasso, we need to specify the structure ahead of time. For some problems, we don’t know the group structure, and yet we would still like highly correlated coefficients to be treated as an implicit group.

Elastic net can achieve this, named after being “like a stretchable fishing net that retains all the big fish”. It has the following objective:

L(w,λ1,λ2)=∣∣y−Xw∣∣22+λ1∣∣w∣∣1+λ2∣∣w∣∣22\mathcal{L}(\bold{w},\lambda_1,\lambda_2)=||\bold{y}-X\bold{w}||^2_2+\lambda_1 ||\bold{w}||_1+ \lambda_2 ||\bold{w}||_2^2L(w,λ1​,λ2​)=∣∣y−Xw∣∣22​+λ1​∣∣w∣∣1​+λ2​∣∣w∣∣22​

This is strictly convex (assuming λ2>0)\lambda_2 >0)λ2​>0), so there is a unique global minimum, even if XXX is not full rank.

It can be shown that any strictly convex penalty on w\bold{w}w will exhibit a group effect, and coefficients of highly correlated variables will be the same.

If two features are identically equal, so X:,j=X:,kX_{:,j}=X_{:, k}X:,j​=X:,k​, one can show that their estimate w^j=w^k\hat{w}_j=\hat{w}_kw^j​=w^k​.

By contrast with Lasso, we might have w^j=0\hat{w}_j=0w^j​=0, and w^k≠0\hat{w}_k \neq 0w^k​=0 and vice versa. Resulting in less stable estimates.

Also, when N<DN<DN<D, lasso can only select up to NNN non-zero coefficients, while elastic net doesn’t have such limits, thus exploring more subsets of variables.

11.4.9 Optimization algorithms

Below, we expand on the method used to solve the lasso problem and other ℓ1\ell_1ℓ1​-regularized convex objectives.

11.4.9.1 Coordinate descent

Sometimes, it’s easier to optimize all variables independently. We can solve for the jjjth coefficient while keeping all others constant.

wj∗=arg min⁡ηL(w+ηej)w^*_j=\argmin_{\eta}\mathcal{L}(\bold{w}+\eta \bold{e}_j)wj∗​=ηargmin​L(w+ηej​)

We can either loop through all variables, select them at random, or update the steepest gradient. This approach has been generalized to GLM with the glmnet library.

11.4.9.2 Projected gradient descent

We convert the non-differentiable ℓ1\ell_1ℓ1​ penalty into a smooth regularizer. We use the split variable trick

w=w+−w−\bold{w}=\bold{w}_+-\bold{w}_-w=w+​−w−​

where w+=max⁡(w,0)\bold{w}_+=\max(\bold{w},0)w+​=max(w,0) and w−=−min⁡(w,0)\bold{w}_-=-\min(\bold{w},0)w−​=−min(w,0)

Thus we get the smooth but constrained optimization problem:

min⁡w+≥0,w−≥0NLL(w+−w−)+λ∑d=1D∣wd++wd−∣\min_{\bold{w}^+\geq 0,\bold{w}^-\geq 0}\mathrm{NLL}(\bold{w}^+-\bold{w}^-)+\lambda\sum_{d=1}^D |w_d^++w_d^-|w+≥0,w−≥0min​NLL(w+−w−)+λd=1∑D​∣wd+​+wd−​∣

In the case of Gaussian likelihood, the NLL is the least square loss, and the objective becomes a quadratic program (see section 8.5.4).

One way to solve this is to use projected gradient descent (see section 8.6.1). We can enforce the constraint by projecting onto the positive orthant, using wd=max⁡(wd,0)w_d=\max(w_d,0)wd​=max(wd​,0). This operation is denoted P+P_+P+​.

The projected gradient update takes the form:

[wt+1+wt+1−]=P+([wt+−∇ηtNLL(wt+−wt−)+ηtλewt−+∇ηtNLL(wt+−wt−)+ηtλe])\begin{bmatrix}\bold{w}_{t+1}^+ \\ \bold{w}_{t+1}^- \end{bmatrix} = P_+\Big( \begin{bmatrix}\bold{w}_t^+ - \nabla \eta_t\mathrm{NLL}(\bold{w}_t^+-\bold{w}_t^-)+\eta_t\lambda\bold{e} \\ \bold{w}_t^- + \nabla \eta_t\mathrm{NLL}(\bold{w}_t^+-\bold{w}_t^-)+\eta_t\lambda\bold{e} \end{bmatrix} \Big)[wt+1+​wt+1−​​]=P+​([wt+​−∇ηt​NLL(wt+​−wt−​)+ηt​λewt−​+∇ηt​NLL(wt+​−wt−​)+ηt​λe​])

where e\bold{e}e is the unit vector of all ones.

11.4.9.3 Proximal gradient descent

Proximal gradient descent can be used to optimize smooth function with non-smooth penalty, like the ℓ1\ell_1ℓ1​.

In section 8.6.2, we showed that the proximal gradient for the ℓ1\ell_1ℓ1​ penalty corresponds to the soft thresholding. Thus the update is:

wt+1=SoftTreshold(wt−ηt∇NLL(wt),ηtλ)\bold{w}_{t+1}=\mathrm{SoftTreshold}(\bold{w}_t-\eta_t\nabla\mathrm{NLL}(\bold{w}_t),\eta_t\lambda)wt+1​=SoftTreshold(wt​−ηt​∇NLL(wt​),ηt​λ)

Where the soft threshold is applied element-wise. This is called the iterative soft threshold algorithm (ITSA). If we combine it we Nesterov momentum, we get the fast ISTA or FISTA method, which is widely used to fit sparse linear models.

11.4.9.4 LARS

The Least Angle Regression and Shrinkage (LARS) generates a set of solutions for different λ\lambdaλ, starting with an empty set up to a full regularization path.

The idea is that we can quickly compute w^(λk)\hat{w}(\lambda_k)w^(λk​) from w^(λk−1)\hat{w}(\lambda_{k-1})w^(λk−1​) if λk≈λk−1\lambda_k\approx \lambda_{k-1}λk​≈λk−1​, this is known as warm starting.

If we only want a single value λ∗\lambda_*λ∗​, it can be more efficient to start from λmax\lambda_{max}λmax​ down to λ∗\lambda_*λ∗​. This is known as the continuation method or homotopy.

LARS starts with a large λ\lambdaλ that only selects the feature that is the most correlated to yyy. Then, λ\lambdaλ is decreased until finding a feature with a correlation of the same magnitude as the first, but with the residual, defined as:

rk=y−X:,Fkwk\bold{r}_k=\bold{y}-X_{:, F_k}\bold{w}_krk​=y−X:,Fk​​wk​

where FkF_kFk​ is the active set of features at step kkk.

Remarkably, we can find the value of λ\lambdaλ analytically, by using a geometric argument (hence “least angle”). This allows the algorithm to quickly jump where the next point of the regularization path where the active set changes.

This repeats until all variables are added.
