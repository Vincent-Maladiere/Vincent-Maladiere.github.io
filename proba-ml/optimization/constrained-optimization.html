8.5 Constrained Optimization

We consider the following problem:

θ∗=arg min⁡θ∈CL(θ)\theta^*=\argmin_{\theta \in \mathcal{C} }\mathcal{L}(\theta)θ∗=θ∈Cargmin​L(θ)

where the feasible set is:

C={θ∈RD:hi(θ)=0,i∈E, gj(θ)≤0,j∈I}\mathcal{C}= \{\theta \in \mathbb{R}^D:h_i(\theta)=0, i \in \mathcal{E},\space g_j(\theta)\leq 0,j\in \mathcal{I}\}C={θ∈RD:hi​(θ)=0,i∈E, gj​(θ)≤0,j∈I}

with E\mathcal{E}E is the set of equality constraints and I\mathcal{I}I the set of inequality constraints.

8.5.1 Lagrange Multipliers

Let’s have a single constrained h(θ)=0h(\theta)=0h(θ)=0.

We know that the gradient of the loss and the gradient of the constraint are (anti-?)parallel at some point θ∗\theta^*θ∗ minimizing L(θ)\mathcal{L}(\theta)L(θ).

  • proof

    If we move by ϵ\epsilonϵ on the constraint surface we have:

    h(θ+ϵ)≈h(θ)+ϵ⊤∇h(θ)h(\theta+\epsilon) \approx h(\theta)+\epsilon^\top \nabla h(\theta)h(θ+ϵ)≈h(θ)+ϵ⊤∇h(θ)

    As we have h(θ+ϵ)=h(θ)=0h(\theta+\epsilon)=h(\theta)=0h(θ+ϵ)=h(θ)=0, we must have ϵ⊥∇h(θ)\epsilon \perp \nabla h(\theta)ϵ⊥∇h(θ).

    We seek a point θ∗\theta^*θ∗ on the constrained surface minimizing L(θ)\mathcal{L}(\theta)L(θ). ∇L(θ)\nabla \mathcal{L}(\theta)∇L(θ) must also be orthogonal to the constraint surface, otherwise we could decrease the objective by just moving on the surface.

h(θ∗)=λL(θ∗)h(\theta^*)=\lambda \mathcal{L}(\theta^*)h(θ∗)=λL(θ∗)

for some λ∈R\lambda\in \mathbb{R}λ∈R.

This leads to the objective:

L(θ,λ)≜L(θ)+λh(θ)L(\theta,\lambda) \triangleq \mathcal{L}(\theta)+\lambda h(\theta)L(θ,λ)≜L(θ)+λh(θ)

At a stationary point, we have:

∇θ,λL(θ,λ)=0  ⟺  λ∇θh(θ)=∇L(θ),h(θ)=0\nabla_{\theta,\lambda}L(\theta,\lambda)=0 \iff \lambda\nabla_\theta h(\theta)=\nabla \mathcal{L(\theta)},h(\theta)=0∇θ,λ​L(θ,λ)=0⟺λ∇θ​h(θ)=∇L(θ),h(θ)=0

This is called a critical point.

If we have m>1m>1m>1 constraints, we can add them:

L(θ,λ)=L(θ)+∑i=1mλihi(θ)L(\theta,\lambda)=\mathcal{L}(\theta)+\sum_{i=1}^m \lambda_i h_i(\theta)L(θ,λ)=L(θ)+i=1∑m​λi​hi​(θ)

We end up with D+mD+mD+m equations for D+mD+mD+m variables and can use unconstrained optimization methods to solve this system.

Exemple

We minimize L(θ1,θ2)=θ12+θ22\mathcal{L}(\theta_1,\theta_2)=\theta_1^2+\theta_2^2L(θ1​,θ2​)=θ12​+θ22​, subject to the constraint θ1+θ2=1\theta_1+\theta_2=1θ1​+θ2​=1.

The Lagrangian is:

L(θ1,θ2,λ)=θ12+θ22+λ(θ1+θ2−1)L(\theta_1,\theta_2,\lambda)= \theta_1^2+\theta_2^2+\lambda(\theta_1+\theta_2-1)L(θ1​,θ2​,λ)=θ12​+θ22​+λ(θ1​+θ2​−1)

Therefore we need to solve:

∂L(θ1,θ2,λ)∂θ1=2θ1+λ=0∂L(θ1,θ2,λ)∂θ2=2θ2+λ=0∂L(θ1,θ2,λ)∂λ=θ1+θ2−1=0\begin{align} \frac{\partial L(\theta_1,\theta_2,\lambda)}{\partial \theta_1} &=2\theta_1+ \lambda =0 \\ \frac{\partial L(\theta_1,\theta_2,\lambda)}{\partial \theta_2} &=2\theta_2+ \lambda =0 \\ \frac{\partial L(\theta_1,\theta_2,\lambda)}{\partial \lambda} &= \theta_1+\theta_2-1=0 \end{align}∂θ1​∂L(θ1​,θ2​,λ)​∂θ2​∂L(θ1​,θ2​,λ)​∂λ∂L(θ1​,θ2​,λ)​​=2θ1​+λ=0=2θ2​+λ=0=θ1​+θ2​−1=0​​

8.5.2 The KKT conditions

We introduce inequality constraints g(θ)≤0g(\theta)\leq0g(θ)≤0:

L(θ)=L(θ)+∞I(g(θ)>0)L(\theta)=\mathcal{L}(\theta)+\infin I(g(\theta)>0)L(θ)=L(θ)+∞I(g(θ)>0)

However, this is a discontinuous function that is hard to optimize.

Instead, we can consider μ≥0:\mu\geq0:μ≥0:

L(θ)=L(θ)+μg(θ)L(\theta)=\mathcal{L}(\theta)+\mu g(\theta)L(θ)=L(θ)+μg(θ)

and our optimization problem becomes:

min⁡θmax⁡μ≥0L(θ,μ)\min_\theta \max_{\mu \geq 0} L(\theta,\mu)θmin​μ≥0max​L(θ,μ)

The general Lagrangian becomes:

L(θ,λ,μ)=L(θ)+∑i=1mλihi(θ)+∑j=1nμjgj(θ)L(\theta,\lambda,\mu)=\mathcal{L}(\theta)+\sum_{i=1}^m \lambda_i h_i(\theta) + \sum_{j=1}^n \mu_j g_j(\theta)L(θ,λ,μ)=L(θ)+i=1∑m​λi​hi​(θ)+j=1∑n​μj​gj​(θ)

When L\mathcal{L}L and ggg are convex, then all critical points must satisfy the Karush-Kuhn-Tucker (KKT) conditions:

  • All constraints are satisfied (feasibility)

    g(θ)≤0,h(θ)=0g(\theta)\leq 0,h(\theta)=0g(θ)≤0,h(θ)=0

  • The solution is a stationary point:

    ∇L(θ∗)+∑i=1mλi∇hi(θ)+∑j=1nμj∇gj(θ)=0\nabla \mathcal{L}(\theta^*)+\sum_{i=1}^m \lambda_i \nabla h_i(\theta) + \sum_{j=1}^n \mu_j \nabla g_j(\theta)=0∇L(θ∗)+i=1∑m​λi​∇hi​(θ)+j=1∑n​μj​∇gj​(θ)=0

  • The penalty for the inequality constraints points in the right direction (dual feasibility):

    μ≥0\mu \geq 0μ≥0

  • The Lagrange multiplies pick up any slack in the inequality constraints:

μ⊙g=0\mu \odot g=0μ⊙g=0

If the L\mathcal{L}L is convex and the constraints define a convex set, then the KKT conditions are sufficient to ensure global optimality.

8.5.3 Linear Programming

We optimize a linear function subject to linear constraints, represented as:

min⁡θc⊤θs.t.Aθ≤b, θ≥0\min_\theta c^\top \theta \quad s.t. \quad A\theta \leq b,\space\theta \geq 0θmin​c⊤θs.t.Aθ≤b, θ≥0

The feasible set defines a convex polytope: a convex space defined as an intersection of half-spaces.

8.5.3.1 The simplex algorithm

It can be shown that the optima of an LP occur at the vertex of the polytope defining the feasible set.

The simplex algorithm solves LPs by moving from vertex to vertex, seeking the edge that most improve the objective.

8.5.3.2 Applications

Applications range from business to machine learning, in robust linear regression settings. It is also useful for state estimation in graphical models.
