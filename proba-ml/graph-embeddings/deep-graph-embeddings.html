23.5 Deep Graph Embeddings

We now focus on graph in unsupervised and semi-supervised settings.

23.5.1 Unsupervised graph embeddings

23.5.1.1 Structural deep network embedding (SDNE)

SDNE method uses auto-encoders preserving first and second order node proximity.

It takes a row of the adjacency matrix WWW as input and produces node embeddings Z=ENC(W;ΘE)Z=\mathrm{ENC}(W;\Theta^E)Z=ENC(W;ΘE). Note that this ignores node features XXX.

The SDNE decoder returns W^=DEC(Z;ΘD)\hat{W}=\mathrm{DEC}(Z;\Theta^D)W^=DEC(Z;ΘD), a reconstruction trained to recover the original graph adjacency matrix.

SDNE preserves second order proximity by minimizing the loss:

∣∣(W−W^)I(W>0)∣∣F2+αSNDE∑ijWij∣∣Zi−Zj∣∣22||(W-\hat{W})\mathbb{I}(W>0)||^2_F+\alpha_{\mathrm{SNDE}}\sum_{ij}W_{ij}||Z_i-Z_j||^2_2∣∣(W−W^)I(W>0)∣∣F2​+αSNDE​ij∑​Wij​∣∣Zi​−Zj​∣∣22​

The first term is similar to matrix factorization techniques, except W^\hat{W}W^ is not computed using outer product. The second term is used by distance based shallow embedding methods.

23.5.1.2 (Variational) graph auto-encoders (GAE)

GAE use graph convolutions to learn node embeddings Z=GCN(W,X;ΘE)Z=\mathrm{GCN}(W,X;\Theta^E)Z=GCN(W,X;ΘE).

The decoder is an outer product W^=DEC(Z;ΘD)=ZZ⊤\hat{W}=\mathrm{DEC}(Z;\Theta^D)=ZZ^\topW^=DEC(Z;ΘD)=ZZ⊤

The graph reconstruction term is the sigmoid cross entropy between the true adjacency and the predicted edges similarity scores:

LG,RECON(W,W^;Θ)=−∑ij[Wijlog⁡σ(W^ij)+(1−Wij)log⁡(1−σ(W^ij))]\mathcal{L}_{\mathrm{G,RECON}}(W,\hat{W};\Theta)=-\sum_{ij} \big[W_{ij}\log \sigma(\hat{W}_{ij})+(1-W_{ij})\log (1-\sigma(\hat{W}_{ij}))\big]LG,RECON​(W,W^;Θ)=−ij∑​[Wij​logσ(W^ij​)+(1−Wij​)log(1−σ(W^ij​))]

To avoid computing the regularization term over all possible nodes pairs, GAE uses negative sampling.

Whereas GAE is a deterministic model, the author also introduced variational graph auto-encoders (VGAE).

The embedding ZZZ is modeled as a latent variable with a standard multivariate prior p(Z)=N(Z∣0,I)p(Z)=\mathcal{N}(Z|\bold{0},I)p(Z)=N(Z∣0,I) and a graph convolution is used as the amortized inference network qΦ(Z∣W,X)q_\Phi(Z|W,X)qΦ​(Z∣W,X).

The model is training by minimizing the corresponding negative ELBO:

NELBO=−EqΦ(Z∣W,X)[log⁡p(W∣Z)]+KL(qΦ(Z∣W,X)∣∣p(Z))=LG,RECON(W,W^;Θ)+KL(qΦ(Z∣W,X)∣∣p(Z))\begin{align} \mathrm{NELBO} &= -\mathbb{E}_{q_\Phi(Z|W,X)}[\log p(W|Z)]+\mathrm{KL}(q_\Phi(Z|W,X)||p(Z)) \\ &= \mathcal{L}_{\mathrm{G,RECON}}(W,\hat{W};\Theta)+\mathrm{KL}(q_\Phi(Z|W,X)||p(Z)) \end{align}NELBO​=−EqΦ​(Z∣W,X)​[logp(W∣Z)]+KL(qΦ​(Z∣W,X)∣∣p(Z))=LG,RECON​(W,W^;Θ)+KL(qΦ​(Z∣W,X)∣∣p(Z))​​

23.5.1.3 Iterative generative modeling of graphs (Graphite)

The graphite model extends GAE and VGAE by introducing a more complex decoder, which iterate between pairwise decoding functions and graph convolutions:

W^(k)=Z(k)Z(k)⊤∣∣Z(k)∣∣22+11⊤NZ(k+1)=GCN(W^(k),Z(k))\begin{align} \hat{W}^{(k)} &=\frac{Z^{(k)}{Z^{(k)}}^\top}{||Z^{(k)}||^2_2}+\frac{11^\top}{N} \\ Z^{(k+1)} &=\mathrm{GCN}(\hat{W}^{(k)},Z^{(k)}) \end{align}W^(k)Z(k+1)​=∣∣Z(k)∣∣22​Z(k)Z(k)⊤​+N11⊤​=GCN(W^(k),Z(k))​​

where Z(0)Z^{(0)}Z(0) is the input of the encoder.

This process allows Graphite to learn more expressive decoders. Finally, similar to GAE, graphite can be deterministic or variational.

23.5.1.4 Methods based on contrastive loss

The deep graph infomax (DGI) method is a GAN-like method for creating graph-level embeddings.

Given one or more real (positive) graphs, each with its adjacency matrix W∈RN×NW\in \R^{N\times N}W∈RN×N and nodes features X∈RN×DX\in \R^{N\times D}X∈RN×D, this method creates fake (negative) adjacency matrices W−∈RN−×N−W^{-}\in \R^{N^-\times N^-}W−∈RN−×N− and their features X−∈RN−×DX^- \in \R^{N^-\times D}X−∈RN−×D.

DGI trains:

i) an encoder that processes both real and fake samples, giving:

Z=ENC(X,W;ΘE)∈RN×LZ−=ENC(X−,W−;ΘE)∈RN−×L\begin{align} Z &= \mathrm{ENC}(X,W;\Theta^E)\in \R^{N\times L} \\ Z^- &= \mathrm{ENC}(X^-,W^-;\Theta^E)\in \R^{N^-\times L} \end{align}ZZ−​=ENC(X,W;ΘE)∈RN×L=ENC(X−,W−;ΘE)∈RN−×L​​

ii) A (readout) graph pooling function R:RN×L→RL\mathcal{R}:\R^{N\times L}\rightarrow \R^LR:RN×L→RL

iii) A discriminator function D:RL×RL→[0,1]\mathcal{D}:\R^L \times \R^L \rightarrow [0,1]D:RL×RL→[0,1] which is trained to output

D(Zi,R(Z))≈1D(Zj−,R(Z−))≈0\begin{align} \mathcal{D}(Z_i,\mathcal{R}(Z)) &\approx 1 \\ \mathcal{D}(Z_j^-,\mathcal{R}(Z^-)) &\approx 0 \end{align}D(Zi​,R(Z))D(Zj−​,R(Z−))​≈1≈0​​

for nodes corresponding to given graph i∈Vi\in Vi∈V and fake graph j∈V−j\in V^{-}j∈V−.

The loss is:

min⁡Θ−EX,W∑i=1Nlog⁡D(Zi,R(Z))−EX−,W−∑j=1N−log⁡(1−D(Zj−,R(Z−)))\min_\Theta -\mathbb{E}_{X,W}\sum_{i=1}^N \log \mathcal{D}(Z_i,\mathcal{R}(Z))-\mathbb{E}_{X^-,W^-}\sum_{j=1}^{N^-}\log (1-\mathcal{D}(Z_j^-,\mathcal{R}(Z^-)))Θmin​−EX,W​i=1∑N​logD(Zi​,R(Z))−EX−,W−​j=1∑N−​log(1−D(Zj−​,R(Z−)))

where Θ\ThetaΘ contains ΘE\Theta^EΘE and the parameter of R\mathcal{R}R and D\mathcal{D}D.

In the first expectation, DGI samples from the real graphs. If only one graph is given, it could sample some subgraphs from it (connected components).

The second expectation sample fake graphs. In DGI, fake graphs use the real adjacency matrix W−=WW^-=WW−=W but fakes features X−X^{-}X− are row-wise random permutation of the real XXX.

The encoder used in DGI is a GCN, though any GNN can be used.

The readout R\mathcal{R}R summarizes an entire (variable size) graph to a single (fixed-dimension) vector. DGI uses a row-wise mean, though other graph pooling might be used, e.g. ones aware of the adjacency.

The optimization of the loss maximize a lower-bound on the mutual information between the outputs of the encoder and the graph pooling function, i.e. between individual node representation and the graph representation.

Graphical Mutual Information (GMI) is a variant that maximizes the MI between the representation of a node and its neighbors, rather than the maximizing the MO of node information and the entire graph.

23.5.2 Semi-supervised graph embeddings

We now discuss semi-supervised losses for GNNs. We consider the simple special case in which we use a nonlinear encoder of the node features XXX, but ignores the graph structure WWW:

Z=ENC(X;ΘE)Z=\mathrm{ENC}(X;\Theta^E)Z=ENC(X;ΘE)

23.5.2.1 SemiEmb

Semi-supervised embeddings (SemiEmb) use an MLP for the encoder of XXX. For the decoder, we can use a distance-based graph decoder:

W^ij=DEC(Z;ΘD)ij=∣∣Zi−Zj∣∣2\hat{W}_{ij}=\mathrm{DEC}(Z;\Theta^D)_{ij}=||Z_i-Z_j||^2W^ij​=DEC(Z;ΘD)ij​=∣∣Zi​−Zj​∣∣2

23.5.2.2 Planetoid

Unsupervised skip-gram methods like DeepWalk and node2vec learn embeddings in a multi-step pipeline, where random walks are first generated from the graph and then used to learn embeddings.

These embeddings are likely not optimal for downstream classification tasks. The Planetoid method extends such random walk methods to leverage node label information during the embedding.

Planetoid first maps nodes to embeddings using a neural net (again ignoring graph structure):

Z=[Zc∣∣ZF]=ENC(X;ΘE)Z=[Z^c||Z^F]=\mathrm{ENC}(X;\Theta^E)Z=[Zc∣∣ZF]=ENC(X;ΘE)

The node embeddings ZcZ^cZc capture structural information while the node embeddings ZFZ^FZF capture feature information.

There are two variant of encoding for ZcZ^cZc:

  • A transductive version that directly learn ZcZ^cZc as an embedding lookup
  • An inductive model where ZcZ^cZc is computed with parametric mappings that act on input features XXX

The planetoid objective contains both a supervised loss and a graph regularization loss. This latter term measures the ability to predict context using nodes embeddings:

LG,RECON(W,W^;Θ)=−E(i,j,γ)log⁡σ(γW^ij)\mathcal{L}_{\mathrm{G,RECON}}(W,\hat{W};\Theta)=-\mathbb{E}_{(i,j,\gamma)}\log \sigma(\gamma \hat{W}_{ij})LG,RECON​(W,W^;Θ)=−E(i,j,γ)​logσ(γW^ij​)

with W^ij=Zi⊤Zj\hat{W}_{ij}=Z^\top_iZ_jW^ij​=Zi⊤​Zj​ and γ∈{−1,1}\gamma\in\{-1,1\}γ∈{−1,1}, with γ=1\gamma=1γ=1 if (vi,vj)∈E(v_i,v_j)\in E(vi​,vj​)∈E is a positive pair and γ=−1\gamma=-1γ=−1 if (vi,vj)(v_i,v_j)(vi​,vj​) is a negative pair.

The distribution under the expectation is defined through a sampling process.

The supervised loss in Planetoid is the NLL of predicting the correct labels:

LSUPL(yN,y^N;Θ)=−1∣VL∣∑i∣vi∈VL∑1≤k≤CyikNlog⁡y^ikN\mathcal{L}_{\mathrm{SUP}}^L(y^N,\hat{y}^N;\Theta)=-\frac{1}{|V_L|}\sum_{i|v_i\in V_L}\sum_{1\leq k\leq C}y_{ik}^N \log \hat{y}_{ik}^NLSUPL​(yN,y^​N;Θ)=−∣VL​∣1​i∣vi​∈VL​∑​1≤k≤C∑​yikN​logy^​ikN​

where iii is a node’s index, and y^iN\hat{y}_i^Ny^​iN​ are computed using a neural net followed by a softmax activation, mapping ZiZ_iZi​ to predicted labels.
