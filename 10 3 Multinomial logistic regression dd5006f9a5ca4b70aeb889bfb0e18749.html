10.3.8 Handling large number of classes

In this section, we discuss the issues that arise when there is a large number of potential classes, like in NLP.

10.3.8.1 Hierarchical softmax

In regular softmax, computing the normalizing term can become the bottleneck since it has a complexity of O(C)O(C)O(C)﻿.

However, by structuring the labels as a tree, we can compute any labels in O(log⁡C)O(\log C)O(logC)﻿ times, by multiplying the probabilities of each edge from the root to the leaf.

A good way to structure it is to use Huffman encoding, with the most frequent labels on top.
