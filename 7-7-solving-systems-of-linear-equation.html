We can formulate linear equations in the form of Ax=bA\bold{x}=\bold{b}Ax=b﻿.
If we have mmm﻿ equations and nnn﻿ unknowns, then AAA﻿ will be a m×nm\times nm×n﻿ matrix and b\bold{b}b﻿ a m×1m\times 1m×1﻿ vector.
If m=nm=nm=n﻿ and AAA﻿ is full rank, there is a single unique solution
If m<nm<nm<n﻿ the system is underdetermined so there is no unique solution
If m>nm>nm>n﻿ the system is overdetermined, not all the lines intersect at the same point

7.7.1 Solving square systems

When m=nm=nm=n﻿, we can solve the LU decomposition for x\bold{x}x﻿.
Ax=bLUx=bUx=L−1b≜yx=U−1y\begin{align} A\bold{x}&=\bold{b} \\ LU\bold{x}&=\bold{b} \\ U\bold{x}&=L^{-1}\bold{b}\triangleq\bold{y} \\ \bold{x}&=U^{-1}\bold{y} \end{align}AxLUxUxx​=b=b=L−1b≜y=U−1y​​
The point is that LLL﻿ and UUU﻿ are triangular, so we don’t need to compute the inverse and can use back substitution instead. For L−1b=yL^{-1}\bold{b}=\bold{y}L−1b=y﻿​
We start by solving y1=b1/L11y_1=b_1/L_{11}y1​=b1​/L11​﻿, then we substitute it into L12y1+L22y2=b2L_{12}y_1+L_{22}y_2=b_2L12​y1​+L22​y2​=b2​﻿ and iterate.
Once we have y\bold{y}y﻿, we can apply the same computation for x=U−1yx=U^{-1}\bold{y}x=U−1y﻿.
