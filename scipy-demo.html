The url above links to my tutorial about scipy linalg and optimization modules, first introduced during EuroScipy 2022 at Basel.
It is a fairly beginner-friendly tutorial, addressed to python programmers interested in getting to know more about some of the core operations in machine learning: matrix operations and optimizers.
The tutorial is made interactive thanks to Jupyter Lite, no local setup is required. Because scripts run directly in the browser through WebAssembly and Piodyde, python packages are downloaded during the launch of the kernel and that might cause some latencies.
See the poster of the tutorial:
https://pretalx.com/euroscipy-2022/talk/ZQJPNY/
