11.2.1 Terminology

We consider models of the form:
p(y∣x,θ)=N(y∣w0+w⊤x,σ2)p(y|\bold{x},\theta)=\mathcal{N}(y|w_0+\bold{w}^\top\bold{x},\sigma^2)p(y∣x,θ)=N(y∣w0​+w⊤x,σ2)
The vector w1:D\bold{w}_{1:D}w1:D​﻿ is known as weights. Each coefficient wdw_dwd​﻿ specifies the change in output we expect if we change the corresponding input xdx_dxd​﻿ by 1 unit.
w0w_0w0​﻿ is the bias and corresponds to the output when all inputs are 0. This captures the unconditional mean of the response w0=E[y]w_0=\mathbb{E}[y]w0​=E[y]﻿​
By assuming x\bold{x}x﻿ to be written as [1,x1,…,xD][1,x_1,\dots,x_D][1,x1​,…,xD​]﻿, we can absorb the term w0w_0 w0​﻿ in w\bold{w}w﻿.
If the input is multi-dimensional y∈Rj\bold{y}\in\mathbb{R}^jy∈Rj﻿, this is called multivariate linear regression:
p(y∣x,W)=∏j=1JN(yj∣wj⊤x,σj2)p(y|\bold{x},W)=\prod_{j=1}^J\mathcal{N}(y_j|\bold{w}_j^\top \bold{x},\sigma_j^2)p(y∣x,W)=j=1∏J​N(yj​∣wj⊤​x,σj2​)
Generally, a straight line will underfit most datasets, but we can apply nonlinear transformations to the input features x′=ϕ(x)\bold{x}'=\phi(\bold{x})x′=ϕ(x)﻿.
As long as the parameters of ϕ\phiϕ﻿ are fixed, the model remains linear in the parameters, even if it’s not linear in the raw inputs.
Polynomial expansion is an example of non-linear transformation: if the input is 1d and we use an expansion of degree ddd﻿, we get ϕ(x)=[1,x,x2,…,xd]\phi(x)=[1,x,x^2,\dots,x^d]ϕ(x)=[1,x,x2,…,xd]﻿​

11.2.2 Least square estimation

To fit a linear regression model to the data, we minimize the negative log-likelihood on the training set:
NLL(w,σ2)=−∑n=1Nlog⁡[(12πσ2)1/2exp⁡(−12σ2(yn−w⊤xn)2)]=N2log⁡(2πσ2)+12σ2∑n=1N(yn−w⊤xn)2\begin{align} \mathrm{NLL}(\bold{w},\sigma^2)&=-\sum_{n=1}^N \log \Bigg[\Big(\frac{1}{2\pi \sigma^2}\Big)^{1/2} \exp\Big(- \frac{1}{2\sigma^2} (y_n-\bold{w}^\top \bold{x}_n)^2\Big)\Bigg]\\ &= \frac{N}{2} \log (2\pi \sigma^2)+\frac{1}{2\sigma^2}\sum_{n=1}^N (y_n-\bold{w}^\top \bold{x}_n)^2 \end{align}NLL(w,σ2)​=−n=1∑N​log[(2πσ21​)1/2exp(−2σ21​(yn​−w⊤xn​)2)]=2N​log(2πσ2)+2σ21​n=1∑N​(yn​−w⊤xn​)2​​
If we only focus on optimizing w\bold{w}w﻿ and not σ2\sigma^2σ2﻿, the NLL is equal to the residual sum of square (RSS):
RSS(w)=12∑n=1n(yn−w⊤xn)2=12∣∣Xw−y∣∣22\mathrm{RSS}(\bold{w})=\frac{1}{2}\sum_{n=1}^n (y_n-\bold{w}^\top \bold{x}_n)^2=\frac{1}{2}||X\bold{w}-\bold{y}||^2_2RSS(w)=21​n=1∑n​(yn​−w⊤xn​)2=21​∣∣Xw−y∣∣22​
11.2.2.1 Ordinary least square
We can show that the gradient is given by:
∇wRSS(w)=X⊤Xw−X⊤y\nabla_\bold{w}\mathrm{RSS}(\bold{w})=X^\top X \bold{w}-X^\top\bold{y}∇w​RSS(w)=X⊤Xw−X⊤y
We set it to zero and obtained the ordinary least square solution:
w^=(X⊤X)−1X⊤y\hat{\bold{w}}=(X^\top X)^{-1}X^\top \bold{y}w^=(X⊤X)−1X⊤y
The quantity X†=(X⊤X)−1X⊤X^\dag =(X^\top X)^{-1}X^\top X†=(X⊤X)−1X⊤﻿ is the left pseudo-inverse of the (non-square) matrix XXX﻿.
We check that the solution is unique by showing that the Hessian is positive definite:
H(w)=X⊤X\mathrm{H}(\bold{w})=X^\top XH(w)=X⊤X
If XXX﻿ is full rank (its columns are linearly independent) then H\mathrm{H}H﻿ is positive definite, hence the least square objective has a global minimum.
11.2.2.2 Geometric interpretation of least squares
We assume N>DN>DN>D﻿, so there are more equations than unknown, the system is over-determined.
We seek a vector y^∈RN\hat{\bold{y}}\in\mathbb{R}^Ny^​∈RN﻿ that lies in the linear subspace spanned by XXX﻿, as close as possible to y\bold{y}y﻿:
arg min⁡y^∈span({x:,1,…,x:,d})∣∣y−y^∣∣2\argmin_{\hat{\bold{y}}\in \mathrm{span}(\{\bold{x}_{:,1},\dots,\bold{x}_{:,d}\})} ||\bold{y-\hat{y}}||_2y^​∈span({x:,1​,…,x:,d​})argmin​∣∣y−y^​∣∣2​
Since y^∈span(X)\bold{\hat{y}}\in\mathrm{span}(X)y^​∈span(X)﻿, there exists some weight vector w\bold{w}w﻿ such that:
y^=Xw\bold{\hat{y}}=X\bold{w}y^​=Xw
To minimize the norm of the residual y−y^\bold{y-\hat{y}}y−y^​﻿, we want the residual to be orthogonal to every column of XXX﻿:
X⊤(y−Xw)=0⇒w=(X⊤X)−1X⊤yX^\top (\bold{y}-X\bold{w})=\bold{0}\Rightarrow \bold{w}=(X^\top X)^{-1}X^\top \bold{y}X⊤(y−Xw)=0⇒w=(X⊤X)−1X⊤y
Hence, our projected value is given by:
y^=X(X⊤X)−1X⊤y\bold{\hat{y}}=X(X^\top X)^{-1}X^\top \bold{y}y^​=X(X⊤X)−1X⊤y
11.2.2.3 Algorithmic issues
Even if we can theoretically invert X⊤XX^\top XX⊤X﻿ to compute the pseudo-inverse X†X^\dagX†﻿, we shouldn’t do it since X⊤XX^\top XX⊤X﻿ could be ill-conditioned or singular.
A more general approach is to compute the pseudo-inverse using SVD. scikit-learn.linear_model uses scipy.linalg.lstsq, which in turn uses DGELSD, which is an SVD-based solver implemented by the Lapack library, written in Fortran.
However, if N≫DN\gg DN≫D﻿, it can be faster to use QR decomposition:
(QR)w=yQ⊤QRw=Q⊤yw=R−1(Q⊤y)\begin{align} (QR)\bold{w}&=y \\ Q^\top QR\bold{w}&=Q^\top y\\ \bold{w}&=R^{-1}(Q^\top y) \end{align}(QR)wQ⊤QRww​=y=Q⊤y=R−1(Q⊤y)​​
Since RRR﻿ is upper triangular, we can solve it using back substitution, avoiding matrix inversion.
An alternative to matrix decomposition is to use an iterative solver, such as the conjugate gradient method (assuming XXX﻿ is symmetric positive definite) and the GMRES (generalized minimal residual method) in the general case.
11.2.2.4 Weighted least squares
In some cases, we want to associate a weight with each example. For example, in heteroskedastic regression, the variance depends on the input:
p(y∣x,θ)=N(y∣w⊤x,σ(x))p(y|\bold{x},\theta)=\mathcal{N}(y|\bold{w}^\top \bold{x},\sigma(\bold{x}))p(y∣x,θ)=N(y∣w⊤x,σ(x))
thus:
p(y∣x,θ)=N(y∣Xw,Λ−1)p(\bold{y}|\bold{x},\theta)=\mathcal{N}(\bold{y}|X\bold{w},\Lambda^{-1})p(y∣x,θ)=N(y∣Xw,Λ−1)
where Λ=diag(1/σ2(xn))\Lambda=\mathrm{diag}(1/\sigma^2(\bold{x}_n))Λ=diag(1/σ2(xn​))﻿.
One can show that the MLE is given by:
w^=(X⊤ΛX)−1X⊤Λy\bold{\hat{w}}=(X^\top \Lambda X)^{-1}X^\top \Lambda\bold{y}w^=(X⊤ΛX)−1X⊤Λy

11.2.3 Other approaches to computing the MLE

11.2.3.1 Solving for offset and slope separately
Above, we computed (w0,w)(w_0,\bold{w})(w0​,w)﻿ at the same time by adding a constant 1\bold{1}1﻿ column to XXX﻿.
Alternatively, we can solve for w0w_0w0​﻿ and w\bold{w}w﻿ separately:
w^=(Xc⊤Xc)−1Xc⊤yc=[∑n=1N(xn−xˉ)(xn−xˉ)⊤]−1[∑n=1N(xn−xˉ)(yn−yˉ)]w^0=yˉ−xˉ⊤w^\begin{align} \bold{\hat{w}}&=(X^\top_cX_c)^{-1}X^\top_c \bold{y}_c \\ &= \Big[\sum_{n=1}^N (\bold{x}_n-\bar{\bold{x}})(\bold{x}_n-\bar{\bold{x}})^\top\Big]^{-1}\Big[\sum_{n=1}^N (\bold{x}_n-\bar{\bold{x}})(\bold{y}_n-\bar{\bold{y}}) \Big] \\ \hat{w}_0&= \bar{\bold{y}} -\bar{\bold{x}}^\top\bold{\hat{w}} \end{align}w^w^0​​=(Xc⊤​Xc​)−1Xc⊤​yc​=[n=1∑N​(xn​−xˉ)(xn​−xˉ)⊤]−1[n=1∑N​(xn​−xˉ)(yn​−yˉ​)]=yˉ​−xˉ⊤w^​​
Thus, we can first estimate w^\hat{\bold{w}}w^﻿ on centered data, then compute w0w_0w0​﻿.
11.2.3.2 1d linear regression
In the case of 1d inputs, the above formula reduces to:
w^1=∑n=1N(xn−xˉ)(yn−yˉ)∑n=1N(xn−xˉ)2=CxyCxxw^0=yˉ−w^1xˉ=E[y]−w^1E[x]\begin{align} \hat{w}_1&=\frac{\sum_{n=1}^N (x_n-\bar{x})(y_n-\bar{y})}{\sum_{n=1}^N (x_n-\bar{x})^2}=\frac{C_{xy}}{C_{xx}} \\ \hat{w}_0 &= \bar{y}-\hat{w}_1 \bar{x} =\mathbb{E}[y]-\hat{w}_1\mathbb{E}[x] \end{align} w^1​w^0​​=∑n=1N​(xn​−xˉ)2∑n=1N​(xn​−xˉ)(yn​−yˉ​)​=Cxx​Cxy​​=yˉ​−w^1​xˉ=E[y]−w^1​E[x]​​
where Cxy=Cov[X,Y]C_{xy}=\mathrm{Cov}[X,Y]Cxy​=Cov[X,Y]﻿ and Cxx=Cov[X,X]=V[X]C_{xx}=\mathrm{Cov}[X,X]=\mathbb{V}[X]Cxx​=Cov[X,X]=V[X]﻿​
11.2.3.3 Partial regression
We can compute the regression coefficient in the 1d case as:
RXY≜∂∂xE[Y∣X=x]=w^1R_{XY}\triangleq \frac{\partial }{\partial x}\mathbb{E}[Y|X=x]=\hat{w}_1RXY​≜∂x∂​E[Y∣X=x]=w^1​
Consider now the 2d case: Y=w0+w1X1+w2X2+ϵY=w_0+w_1 X_1 +w_2 X_2+\epsilonY=w0​+w1​X1​+w2​X2​+ϵ﻿, where E[ϵ]=0\mathbb{E}[\epsilon]=0E[ϵ]=0﻿​
The optimal regression coefficient for w1w_1w1​﻿ is RYX1.X2R_{Y X_1.X_2}RYX1​.X2​​﻿ where we keep X2X_2X2​﻿ constant:
w1=RY:X1.X2=∂∂xE[Y∣X1=x,X2]w_1=R_{Y:X_1.X_2}=\frac{\partial}{\partial x}\mathbb{E}[Y|X_1=x,X_2]w1​=RY:X1​.X2​​=∂x∂​E[Y∣X1​=x,X2​]
We can extend this approach to all coefficients, by fixing all inputs but one.
11.2.3.4 Online computing of the MLE
We can update the means online:
xˉ(n+1)=xˉ(n)+1n+1(xn+1−xˉ(n))\bar{x}^{(n+1)}=\bar{x}^{(n)}+\frac{1}{n+1}(x_{n+1}-\bar{x}^{(n)})xˉ(n+1)=xˉ(n)+n+11​(xn+1​−xˉ(n))
One can show that we can also update the covariance online as:
Cxy(n+1)=1n+1[xn+1yn+1+nCxy(n)+nxˉ(n)yˉ(n)−(n+1)xˉ(n+1)yˉ(n+1)]C_{xy}^{(n+1)}=\frac{1}{n+1}[x_{n+1} y_{n+1}+nC_{xy}^{(n)}+n\bar{x}^{(n)}\bar{y}^{(n)}-(n+1)\bar{x}^{(n+1)}\bar{y}^{(n+1)}]Cxy(n+1)​=n+11​[xn+1​yn+1​+nCxy(n)​+nxˉ(n)yˉ​(n)−(n+1)xˉ(n+1)yˉ​(n+1)]
We find similar results for Cxx(n+1)C_{xx}^{(n+1)}Cxx(n+1)​﻿.
To extend this approach to DDD﻿ dimensions inputs, the easiest approach is SGD. The resulting algorithm is called least mean square.
11.2.3.5 Deriving the MLE from a generative perspective
Linear regression is a discriminative model of the form p(y∣x)p(y|\bold{x})p(y∣x)﻿, but we can also use generative models for regression.
The goal is to compute the conditional expectation:
f(x)=E[y∣x]=∫y p(y∣x)dy=∫y p(x,y)dy∫p(x,y)dyf(\bold{x})=\mathbb{E}[y|\bold{x}]=\int y\,p(y|\bold{x})dy=\frac{\int y\,p(\bold{x},y)dy}{\int p(\bold{x},y)dy}f(x)=E[y∣x]=∫yp(y∣x)dy=∫p(x,y)dy∫yp(x,y)dy​
Suppose we fit p(x,y)p(\bold{x},y)p(x,y)﻿ using a MVN. The MLEs for the parameters of the joint distribution are the empirical means and covariance:
μx=1N∑nxnμy=1N∑nynΣxx=1N∑n(xn−μx)(xn−μx)⊤=1NXc⊤XcΣxy=1N∑n(xn−μx)(yn−μy)⊤=1NXc⊤yc\begin{align} \mu_x&=\frac{1}{N}\sum_n \bold{x}_n \\ \mu_y&=\frac{1}{N}\sum_n y_n \\ \Sigma_{xx}&=\frac{1}{N}\sum_n (\bold{x}_n -\mu_x)(\bold{x}_n-\mu_x)^\top = \frac{1}{N} X_c^\top X_c \\ \Sigma_{xy}&=\frac{1}{N}\sum_n (\bold{x}_n -\mu_x)(y_n-\mu_y)^\top = \frac{1}{N} X_c^\top \bold{y}_c \end{align}μx​μy​Σxx​Σxy​​=N1​n∑​xn​=N1​n∑​yn​=N1​n∑​(xn​−μx​)(xn​−μx​)⊤=N1​Xc⊤​Xc​=N1​n∑​(xn​−μx​)(yn​−μy​)⊤=N1​Xc⊤​yc​​​
Hence we have:
E[y∣x]=μy+Σxy⊤Σxx−1(x−μx)\mathbb{E}[y|\bold{x}]=\mu_y+\Sigma_{xy}^\top\Sigma_{xx}^{-1}(\bold{x}-\mu_x)E[y∣x]=μy​+Σxy⊤​Σxx−1​(x−μx​)
which we can rewrite as:
E[y∣x]=w0+w⊤x\mathbb{E}[y|\bold{x}]=w_0+\bold{w}^\top \bold{x}E[y∣x]=w0​+w⊤x
using w=Σxy⊤Σxx−1\bold{w}=\Sigma_{xy}^\top \Sigma^{-1}_{xx}w=Σxy⊤​Σxx−1​﻿​
Thus we see that fitting the joint model and then conditioning it is the same as fitting the conditional model. However, this is only true for Gaussian models.

11.2.4 Measuring goodness of a fit

11.2.4.1 Residuals plot
We can check the fit of the model by plotting the residuals rn=yn−y^nr_n=y_n-\hat{y}_nrn​=yn​−y^​n​﻿ vs the input xn\bold{x}_nxn​﻿, for a chosen dimension.
The residual model assumes a N(0,σ2)\mathcal{N}(0,\sigma^2)N(0,σ2)﻿ distribution, therefore the residual plot should show a cloud of points around 0 without any obvious trend. This is why b) above is a better fit than a).
We can also plot the prediction y^\hat{\bold{y}}y^​﻿ against y\bold{y}y﻿, which should show a diagonal line.
11.4.2.2 Prediction accuracy and R2
We can assess the fit quantitatively using the RSS or the RMSE:
RSS=∑n(yn−w⊤xn)2RMSE≜1NRSS(w)\begin{align} \mathrm{RSS}&=\sum_n (y_n-\bold{w}^\top \bold{x}_n)^2\\ \mathrm{RMSE}&\triangleq\frac{1}{N}\sqrt{RSS(\bold{w})} \end{align}RSSRMSE​=n∑​(yn​−w⊤xn​)2≜N1​RSS(w)​​​
The R2, or coefficient of determination is a more interpretable measure:
R2≜1−∑n(y^n−yn)2∑n(yˉ−yn)2=1−RSSTSS≤1R^2\triangleq 1-\frac{\sum_n (\hat{y}_n-y_n)^2}{\sum_n(\bar{y}-y_n)^2}=1-\frac{\mathrm{RSS}}{\mathrm{TSS}}\leq 1R2≜1−∑n​(yˉ​−yn​)2∑n​(y^​n​−yn​)2​=1−TSSRSS​≤1
It measures the variance in the prediction relative to a simple constant prediction y^n=yˉ\hat{y}_n=\bar{y}y^​n​=yˉ​﻿.
